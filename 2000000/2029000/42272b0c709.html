爪研ぐ猫魂

メテム : よし、アルカディアに臨む心構えができたようだね。
しばらくはワタシがサポートするから、安心してくれたまえ。
メテム : それでは、キミの初戦の相手を紹介しよう。
モニターを見てくれ。
メテム : ライトヘビー級最初の闘士は……
爪研ぐ猫魂（ねこだましい）「ブラックキャット」！
メテム : そう、先ほどキミを値踏みしていた娘だよ。
まだ闘士になって間もないルーキーだが、油断は禁物、
若手の中でも将来を有望視されてる実力者だからね。
メテム : 彼女が注入するのは「キャスパリーグ」の魂……
我らの世界において、古の英雄をも苦戦させたという怪猫だ。
メテム : その魂を駆使するブラックキャットは、
鉄をも切り裂く鋭い爪と、驚異的な俊敏さを獲得している。
素早い動きに翻弄されないよう注意してくれ。
メテム : なお、闘士たちが支配する、リングと呼ばれる闘いの舞台は、
彼らのファイトスタイルやコンセプトに合わせて、
それぞれ固有の環境に設定されている。
メテム : アルカディアは闘技であると同時に、ショーでもあるからね。
派手な演出を施して観客を喜ばせるのさ。
メテム : ところで、ワタシは試合の実況も担当しているんだ。
キミの闘いを最大限に盛り上げるから、
その点も期待してくれたまえ。
メテム : では、記念すべきデビューマッチを、
華々しき勝利で飾れることを祈ってるよ……！
ブラックキャット : 覚悟しな、この爪の餌食になってもらうよ！
メテムの実況 : 初戦の相手は、「ブラックキャット」！
メテムの実況 : 生身の挑戦者は、
怪猫の魂に対抗できるかァ！？
ブラックキャット : この切れ味を楽しみな！
メテムの実況 : 華麗に攻撃をかわしたァ！
ブラックキャット : 私の技を見切れるかな？
ブラックキャット : これでも喰らえ！
ブラックキャット : この爪で切り裂けないものはないよ！
メテムの実況 : 出るぞ、ブラックキャットの必殺技ァ～！
メテムの実況 : なんと、リングが壊れてしまったぞォ～！
メテムの実況 : え～、試合中ですが、
只今、リングの修復を行っています。
ブラックキャット : 空高く打ちあがれッ！
メテムの実況 : なんと、挑戦者がリングアウトだァ～！
メテムの実況 : 挑戦者、なんとか立ち上がったァ～
ブラックキャット : 串刺しにしてやるよ！
ブラックキャット : 切り刻む！
メテムの実況 : 強烈な一撃！
これは効いたかァ～！？
メテムの実況 : 得意技を喰らったァ！
しかし、なんとか耐えている！
メテムの実況 : 立ち上がったァ！
だが、挑戦者はピンチ！
ブラックキャット : おいで、私の分身！
メテムの実況 : 必殺技が炸裂！渾身の一撃だァ～！
メテムの実況 : 実況はメテムでお送りしました！
アルカディアンレフェリー : Ｋ.Ｏ！
メテム : ブラボォ～、ブラボォ～！
素晴らしい勝利だったよ！
メテム : やはり、キミを新たなスターと見定めた、
オーナーの目に狂いはなかったようだ……。
メテム : それでは、初戦を突破したところで、
アルカディアについて、もう少し詳しく説明しておこう。
キミもまだまだ疑問が多いだろうしね。
メテム : この闘技ショーは、キミの同胞……
トライヨラの民を自称する者たちが現れるより、
ずっと前からアレクサンドリアで行われていたんだ。
メテム : ただ、その形態は今とはずいぶんと違った……。
最初は「魂の再現体」同士を闘わせる形式だったのさ。
メテム : そう、キミが先ほど率いて闘った闘士たちと同様の存在だ。
だが魂資源を利用した蘇生が可能になってからは、
より刺激的な「安全な死闘」が行われ始めた。
メテム : さらに、ゾラージャとの連王体制となり、
かの武王の提案で、魔物の魂で肉体を強化する技術が生まれてね。
メテム : アルカディアの闘いもまた変貌を遂げた。
闘士たちは魔物化し、闘いはより派手に、より過激に輝き、
人々もまた熱狂したのさ！
メテム : ところが、ゾラージャが外征に着手した影響で、
アルカディアに保管されていた魂も、軍の管理下に置かれた。
おかげで、しばらく閉鎖を余儀なくされていたのだ。
メテム : だが、武王は乱心の末にキミに斃された。
光明が見えたのも束の間、愛すべき女王までもが喪われ、
今やエバーキープは大混乱だ。
メテム : そんなときに闘技とは、いかがなものかと思うかもしれないが、
悲しみに沈む人々に活力と笑顔を少しでも……。
この理念については前にも話したとおりさ。
メテム : 事実、キミという挑戦者の勝利で観客は大いに沸いていた。
これからも、アメージングな闘いぶりを期待しているよ！
ブラックキャット : アンタ、いったい何者なんだ？
まさか、魔物の魂も使ってないヤツに負けるなんて！
ブラックキャット : ああ～、悔しい！
とてもじゃないけど、このまま黙って帰れない……。
ブラックキャット : ねえ、アンタ、しばらく傍にいさせてもらえない？
試合に負けたことを恨んでるとか、そういうんじゃなくてさ、
その強さに興味が湧いてきたんだよ。
ブラックキャット : まあ、私のことは気にしないで。
ここで、大人しく見てるだけだからさ……。
メテム : やれやれ、ブラックキャットはまだ若く気まぐれなところがある。
どうか無礼を許してやってくれ。
