リングはアトリエ

ヤーナ : さあ、このまま勝ち抜いていこう！
生身の挑戦者の快進撃は始まったばかりだ。
ヤーナ : 次の闘士を紹介するから、モニターを見て。
ヤーナ : クルーザー級、2人目の相手は、
リングはアトリエ「シュガーライオット」！
ヤーナ : 元はグラフィティ・アーティストってやつでね。
ソリューション・ナインの街中にある落書きは、
大抵は彼女の手によるものか、そのスタイルを模倣したものさ。
ヤーナ : プリンの魂を注入することで特殊な塗料を生成して、
スプレーガンで描いたアートを具現化させてくるんだ。
その力で、リングの環境まで変えてしまうんだよ。
ヤーナ : 彼女にとって、リングはアトリエで、闘いは芸術表現ってわけ。
独特な言動も含めて、呑まれたら一巻の終わり……。
何がきても、冷静な判断を心掛けてくれ！
ヤーナ : さあ、この勢いでシュガーライオットを倒して、
クルーザー級王者への階段を駆け上がろう！
ヤーナ : そうそう、遠隔観戦してる観客まで、
シュガーライオットによって徹底してデザインされてるんだ。
余裕があったら外周を見てごらん。
シュガーライオット : どんなインチュピレーチョンをくれるのかちら？
メテムの実況 : さあ、「シュガーライオット」のアトリエへようこそ！
シュガーライオット : さ～て、描きまちゅよ～。
メテムの実況 : 描いたものを具現化する、
「シュガーグラフィティ」だァ～！
シュガーライオット : 何色がおちゅき？
シュガーライオット : イメージが降ってきまちた！
あたちのアートにいらっちゃい。
メテムの実況 : 「ランドスケープ」！
アトリエが砂漠に変化したぞォ～！
シュガーライオット : これに、サンサン描き足ちてっと！
シュガーライオット : さてさて、砂漠といえば……
チクチクでちゅ～！
メテムの実況 : サボテンが針を飛ばしている！？
これは危ない！
シュガーライオット : う～ん、何か物足りまちぇん……
そうだ！
メテムの実況 : 流砂が出現！
飲み込まれたらひとたまりもない！
シュガーライオット : 何色がおちゅき？
シュガーライオット : いっぱい、描いちゃいまちゅ～。
シュガーライオット : こんなのはどうかちら？
メテムの実況 : アメージング！
得意技を見切ったァ～！
シュガーライオット : イメージが湧いてきまちた！
こんなアートはどうかちら？
メテムの実況 : ビューティフル！
アトリエに綺麗な河が流れています！
メテムの実況 : 挑戦者、ダウン！
さァ、立ち上がれるかァ～！？
メテムの実況 : 立ち上がったぞォ！
だが、油断はできない！
シュガーライオット : だめだめ、やっぱりこうでなくっちゃ！
ゴロゴロ、ザーザーっと！
メテムの実況 : 雷雨で河が汚れてしまったァ～！
恐るべし、シュガーライオット！
シュガーライオット : きたきた、イメージが溢れてきまちた！
破壊こそアートでちゅ！
メテムの実況 : 巨大な嵐が出現！
真下にいると大変だァ～！
メテムの実況 : 挑戦者、復活！
巻き返せるかァ～！？
メテムの実況 : オー・マイ・ガッシュ！
挑戦者が倒れたァ～！
シュガーライオット : いっぱい、描いちゃいまちゅ～。
シュガーライオット : こんなのはどうかちら？
メテムの実況 : 余裕で攻撃をかわしたァ！
シュガーライオット : いっぱい、描いちゃいまちゅ～。
アルカディアンレフェリー : Ｋ.Ｏ！
メテムの実況 : 実況はメテムでお送りしました！
ヤーナ : さすがだね！
アンタならやってくれると信じてたよ！
ヤーナ : 次は、いよいよクルーザー級王者が相手だけど……
しばらく時間もあるし、その前にやっておきたいことがある。
ヤーナ : 試合が終わった、ダンシング・グリーンとシュガーライオットに、
魂蝕症のことを伝え、発症してないか調べてあげたいんだ。
彼らにだって、何も知らずに死んでほしくないからね。
ヤーナ : この控室に盗聴器の類がないかは、あらかじめ確認済みさ。
ネユニの端末を借りてきたから、私が検診をするよ。
ヤーナ : そろそろ、来てくれるはずなんだけど……
ダンシング・グリーン : ウェーイ！
もしかして打ち上げのパーティーでも始めるのかい？
シュガーライオット : あなたとの闘いで得たインチュピレーチョンを、
はやくアートに昇華ちたいのに……
なんの用なのかちら？
ヤーナ : 検診の結果、幸いふたりとも発症してなかったけど……
かなり危うい数値が出ちゃってる。
つまり、魂蝕症を防ぐには闘士を続けるべきじゃないってこと。
ダンシング・グリーン : 俄然、気分がサガっちまったな……。
シュガーライオット : にわかには信じられまちぇんが、
あなたもハニー・Ｂも引退ちゅると聞きまちた。
嘘を言っているわけでもなさそうでちゅ。
ヤーナ : 突然、引退しろだなんて受け入れがたいと思う。
でもさ、どうか命の大切さをよく考えて決断してほしい。
ダンシング・グリーン : 決めた、オレは引退するぜ。
ここだけの話、ダンシング・グリーンでいるのはキツかったんだ。
ダンシング・グリーン : パーティー野郎っていうイメージを保つために、
無理してまで、歓楽街を遊び歩かなきゃならなくてさ……。
休日もダンスの練習でヘトヘトだったんだ。
ダンシング・グリーン : なかなか言い出せなかったから、いい機会だったぜ。
ダンスはもともと好きだったんだが……
それはほかでもできるからな。
ヤーナ : 私、ずっとダンシング・グリーンは、
本当にパーティーで遊び明かすのが好きだって思ってた。
結構、大変だったんだね……。
シュガーライオット : アルカディアは、あたちの大事なアトリエでちゅ。
だから、失うのはとても残念……。
シュガーライオット : でも、もとはといえば、あたちの原点はチュトリート。
もちかちたら、初心に返ってアートを見ちゅめ直せっていう、
啓示かもちれまちぇん。
シュガーライオット : あたちにとって壁というキャンバチュさえあれば、
街中、どこでもアトリエでちゅから！
ダンシング・グリーン : それによ、魂を使わず真っ向勝負してくるアンタを見てたらさ……
素の自分ってヤツも、イイもんだなって思えたからな！
ヤーナ : ふたりとも、真剣に考えてくれてありがとう！
シュガーライオット : まだまだ、死にたくありまちぇんからね。
忠告ちてくれて助かりまちた。
ダンシング・グリーン : ここで長く話し込んでると怪しまれそうだな。
どのタイミングでソサエティに意志を伝えるかは、
日を改めて相談しようぜ。
ヤーナ : アンタとの闘いで影響を受けたのはもちろんだけど……
あのふたりも、Ｂセンパイと同じで、
ここ以外に活躍の場を持ってるのが救いになったね。
ヤーナ : それに比べて……
ブルートボンバーやハウリングブレードは、
アルカディアがすべてだって言ってた。
ヤーナ : 彼らは簡単に引退しないだろう。
その気持ちも痛いほどわかるけれどさ……。
