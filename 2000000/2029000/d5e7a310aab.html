ふたり掛かりの相手に、よくやってくれたよ！

すっごくカッコ良かったよ！

ヴァンプ・ファタール : 場外乱闘では、どこまで闘えるのかしらね……。
ここには闘士の魂を実体化する装置もないし、
あの邪魔な助っ人も呼べなくてよ？
ヤーナ : そ、そんな、卑怯だぞ……！
威厳のあるエレダイト族 : 目も当てられぬほどの醜態、闘士の風上にもおけん。
敗者は大人しく去るのみ！
軽薄な男性 : あ、あれは「ザ・タイラント」……！？
アルカディアの現統一王者が、なんでここに……！
ザ・タイラント : 生身の挑戦者よ、武器をしまえ。
真の闘士は神聖なリングでしか闘わぬもの。
ザ・タイラント : 紹介が遅れたな。
私こそがアルカディアの現統一王者、
至天の座の称号を戴く者である。
ザ・タイラント : 貴様はゲームチェンジャーとも呼ばれ、
善玉闘士（ベビーフェイス）のつもりかもしれぬが、
私からすれば、アルカディアを破壊する悪役闘士（ヒール）だ。
強面の男性 : そうだそうだ！
俺たちのアルカディアをぶっ壊す極悪人だ～～～！
ザ・タイラント : 貴様が壊さんとしているもの、
それが如何にして成ってきたか教えてやろう。
心して聴けい！
ザ・タイラント : かつて、リンドブルムという国があった。
我らがオーナーの先祖は、彼の国の貴族であり、
当地で盛んであった演劇を愛する大パトロンであったという。
ザ・タイラント : しかし、雷光大戦が勃発。
崩壊した祖国を脱した彼は荒廃した世界を彷徨い続け……
やがて、敵国であるアレクサンドリアに辿り着いた。
ザ・タイラント : 敵国の者も快く受け入れ、人々が助け合って暮らす様に、
彼は古の理想郷、アルカディアを見たという。
ザ・タイラント : 先祖は考えた、自分にできる恩返しは何かと……。
そして、当時アレクサンドリアにて辛い生活の余興として、
密かに楽しまれていた闘技に目を付けた。
ザ・タイラント : 祖国より持ち出した私財を注ぎ、
闘技と演劇を合わせた闘技ショーを生み出した。
これこそが「アルカディア」の源流である……！
ザ・タイラント : だが、環境が厳しさを増す中で、
次第に闘技ショーの開催頻度は減らさざるを得ず……
やがて忘れ去られていったのだ。
ザ・タイラント : 時は流れ、世代は変わる。
エレクトロープ技術の高まりにより、生活は安定……
魂資源の活用によって安寧を得た人々は、退屈さを感じ始めた。
ザ・タイラント : ここにきて我らがオーナーは、
魂の再現体を闘わせる形でアルカディアを復活させ、
闘技ショーという興奮を、刺激を、活力を与えんとしたのだ。
ザ・タイラント : そして、より激しい闘いを求める人々の求めに応じ、
ついに魔物の魂という力を得て、
アルカディアは国民的な娯楽へと上り詰めたのだよ！
ザ・タイラント : 貴様が破壊しようとしているのは、
かような歴史を、想いを継承してきた神聖な闘いの場……！
ザ・タイラント : よいか、我らが闘技ショーに、魔物の魂は不可欠ッ！
断じて、貴様のようなどこの馬の骨とも知れぬ輩に、
奪われてよいものではないのだッ……！
軽薄な男性 : そのとおりだ～！
変革なんてさせるものか～～～！
物柔らかな女性 : そうよ！
激しさだけじゃなく、真剣勝負のスリルこそが見たいのよ！
ザ・タイラント : 貴様は生身で闘う異端の存在として注目されたが、
所詮は盛り上げるための悪役闘士（ヒール）でしかない。
やがては倒されるのが運命なのだ！
ヤーナ : 見てのとおり、ザ・タイラントは生身のままでも強い。
その上、統一王者として保管庫を自由にできる権利を行使して、
毎回、最適な魔物の魂を選んで使ってくるんだ。
ヤーナ : だから、これだっていうアドバイスはできない。
せめて相手が使う魂がわかればいいんだけど……
ヤーナ : ともあれ、救出作戦の件も気になるところだ。
早くジムに戻ろうか。
ヤーナ : レザラたちはまだ戻ってないようだ。
心配でしょうがないけど、今は待つしかないね……。
ネユニ : ヤーナ姉！
ヤーナ : ネユニ、よかった……！
レザラ : すまない、ユトロープは助け出すことができなかった……。
ハニー・Ｂ・ラブリー : ネユニは来客用の迎賓区画で見つけたんだけど、
ユトロープの姿はなくて、探してる余裕もなかったの。
ネユニ : さらわれたときに……ユト姉はレギュレーターを外されて、
どこか別の場所に連れていかれたんだ。
ヤーナ : そ、それって……引退したとされてきた闘士たちと……
魂蝕症で死んだ人たちと同じ処遇じゃないか！
ダンシング・グリーン : おいおい、みんなしてサガっちまってどうするんだ？
まだ、そうと決まったわけじゃないだろう……。
ダンシング・グリーン : こういうときこそ、気分をアゲていこうぜッ！
シュガーライオット : ちょっとあなた！
いくらなんでも、デリカチーがないでちゅよ！
ヤーナ : いや、ダンシング・グリーンの言うとおりだ……
今はまだ落ち込んでいるときじゃない。
こうして、妹が無事に戻ってきただけでも感謝しないとね。
ヤーナ : ただ、やっぱりユト姉のことは心配だよ。
ネユニを救出したことはオーナーも先刻承知だろうし、
何を仕掛けてくるのか……。
レザラ : 大丈夫、その点なら手を打ってある。
実は今回の作戦には影の協力者がいてね……
レザラ : 驚くなかれ、メテムさ。
レザラ : 無理を承知で掛け合ったんだが、こちら側についてくれてね。
ボクたちの作戦が、すぐにオーナー側へ報告されないように、
時間稼ぎを請け負ってくれたんだ。
レザラ : さらに、メテムはユトロープの行方に心当たりがあるそうで、
必ず彼女を無事に返すと約束してくれたんだよ。
ヤーナ : あのメテムが……見直したよ。
ヤーナ : 不安はあるけど……
ここは、メテムを信じるしかないよ。
ネユニ : そうそう、私からも朗報があるの。
捕まっていた部屋にあった端末をハッキングして、
役立つ情報がないか探っていたとき、いいものを見つけたんだ。
ネユニ : それは、統一王者の戦略データ。
Sakilinaさんとのタイトルマッチでは、
「ベヒーモス」の魂を利用するつもりみたいだよ。
ヤーナ : 助かるよ！
使う魂の種類がわかれば、
Sakilinaも闘いやすくなるはず。
ヤーナ : よし、思ってもみなかった情報まで手に入ったし、
そろそろ試合の準備もできている頃合いだ……
私たちはアルカディア・ソサエティに向かうとしようか。

魔物の魂を使った闘技を終わらせ、
魂蝕症の治療薬で皆を救うことができるはずだ。
レザラ : ここからはメテムを信じるのと、
Sakilinaに頼るほかない。
皆でキミの勝利を祈っているよ……！
レザラ : この子も一緒に行きたそうだな。
名前は「ポム」っていうんだけど、アルカディアに行けば、
ヘクトールと会えると思っているのかもしれない……。
ヤーナ : ポムも、Sakilinaを応援したいのかもよ。
控室まで連れてってあげよう。
シュガーライオット : イメージが湧いてきまちた！
あなたが見事に統一王者になって、
みんな救われ祝福ちゃれている光景でちゅ……！
ダンシング・グリーン : ウェーイ！
タイトルマッチだからって気負うことなんてないぜ、
アンタが思いっきり楽しめば、結果だってついてくるはずさ！
ハニー・Ｂ・ラブリー : アルカディアはみんなが注目する大舞台だからね、
生身の闘技に変革してもらって、アタシも返り咲きたいの。
ここで負けたら、突き刺しちゃうぞ☆
レザラ : ユトロープのことは、メテムを信じるしかない。
どうかキミは、ヘクトールの無念を晴らすためにも、
統一王者になって、魔物の魂を使う闘技を終わらせてくれ！
ネユニ : オーナーがリンドブルム貴族の末裔なら、
魂蝕症の治療薬の存在を知っているというのも頷けますね……。
みんなを治せるよう、ザ・タイラントを倒してください！

ヤーナ : さあ、ポムと一緒にアルカディア・ソサエティの控室に向かおう。
タイトルマッチに臨むアンタの花道だね。
ヤーナ : せっかくだし、通りにいるファンたちに声をかけていこうよ。
きっとアンタを激励してくれるだろうからさ。
ヤーナ : この道を戻るとき、きっとアンタは生身の統一王者だ。
私はそう信じてるよ！
ポム : ニャーニャー！
ポムの鳴き声に、どことなく逸る気持ちを感じる。
早くアルカディア・ソサエティに向かいたいようだ。
興奮した少年 : Sakilinaさん、タイトルマッチがんばって！
絶対にザ・タイラントを倒して、統一王者になってね！
興奮した少年 : ボクもいつかお姉さんみたいな闘士になったら、
一緒にタッグを組んでほしいな。
そして、ふたりで悪い闘士をやっつけるんだ……！
少年の父親 : 息子ともども、あなたの勝利を願ってますよ……！
物柔らかな女性 : Sakilinaさん、ついにタイトルマッチね！

とても光栄だよ、どうもありがとう！

哀しみに沈んでいたときに、強大な相手に生身で挑むキミを見て、

キミが統一王者となってくれたら、いっそう励みになるよ……！
