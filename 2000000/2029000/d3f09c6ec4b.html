孤高の狼

ヤーナ : クルーザー級王者は、王にふさわしい別格の実力を持っている。
でも、私はアンタが勝つと信じてるよ！
ヤーナ : よし、改めて決戦の相手の紹介だ……！
モニターを見て。
ヤーナ : クルーザー級の現王者、
孤高の狼「ハウリングブレード」！
ヤーナ : 彼は、闘士になる前は駆除人をしていてね。
当時から剣の腕前では、並ぶ者がいないといわれるほどで、
その噂を聞いたオーナーの勧誘で、闘士になったらしい。
ヤーナ : 端正な顔立ちだったこともあって、
デビュー直後から善玉闘士として活躍しててさ。
空想物語風のリングや、光剣がトレードマークになってる。
ヤーナ : 注入する魂は、フェンリルと呼ばれる魔狼のもの。
これにより強化された肉体から繰り出される斬撃の数々は、
素早さと鋭さの両方で警戒が必要だよ！
ヤーナ : それじゃあ、Sakilina、
クルーザー級王者を打ち破って、彼を引退させてくれ！
ハウリングブレード : さあ、はじめようじゃないか。
メテムの実況 : クルーザー級王者「ハウリングブレード」の登場です！
ハウリングブレード : 斬る！
ハウリングブレード : この技についてこれるかい？
メテムの実況 : なんとォ～！
あまりの速さに残像が見えている！？
メテムの実況 : 落ち着いて攻撃を避けているぞ！
ハウリングブレード : ちょっと翻弄してやろう。
ハウリングブレード : キミに、これが見破れるかな？
ハウリングブレード : いけッ！
ハウリングブレード : 土の力で…！
ハウリングブレード : さすがだね……。
でも、これを耐えることができるかな？
ハウリングブレード : さあ、その身に受け止めてみろ！
ハウリングブレード : 驚いたよ、まだ闘えるなんてね。
フフフ、楽しくなってきた！
ハウリングブレード : フェンリルの力で、
面白いものを見せてあげるよ！
メテムの実況 : お～っと！
巨大な像が現れた～！
メテムの実況 : 巨大な像が切り倒されてしまったァ～！
ハウリングブレード : もらったァ！
ハウリングブレード : 見破ってみろ
ハウリングブレード : これでどうだァ！
メテムの実況 : 挑戦者、ダウン！
さァ、立ち上がれるかァ～！？
メテムの実況 : 治癒魔法が発動！
起死回生なるかァ～！？
アルカディアンレフェリー : Ｋ.Ｏ！
メテムの実況 : 実況はメテムでお送りしました！
ヤーナ : やってくれたね！
アンタ、本当にとんでもないヤツだ……！
ハウリングブレード : ボクの完敗だよ……
ハウリングブレード : 闘士に二言は無い。
約束どおり引退しよう。
ハウリングブレード : ……これでボクは、命の危機を脱したわけだ。
でも、それと同時に、生きがいを失ってしまった。
ハウリングブレード : これからいったい、何をして生きていけばいいんだろう？
本当に、わからないんだ。
ハウリングブレード : 彼はオーナーに殺されたも同然だし、
興行のために、多くの闘士を犠牲にしてきたことも確かだ。
仇を討つべきだというのはわかるよ。
ハウリングブレード : でも、前に話したとおり、ボクたちはオーナーに恩があってね。
だからこそ、ボクは死に至る病さえ受け入れて、
闘士として闘い続けると決めたんだ。
ヤーナ : 死ぬなんて自分勝手だ！
ブルートボンバーは死にたくなんてなかっただろうし、
ユトロープはアンタを救いたくて、ひとりで闘ってたんだよ！
ハウリングブレード : ユトロープが……なんだって……？
ハウリングブレード : 彼女は、例の症状が出始めた時に、
何も言わずボクの前から、アルカディアからも姿を消したんだ。
ハウリングブレード : そのあと、どれだけ探しても会うことができなかった。
だから、ウィケッドサンダーの特別試合が開催されたとき、
本当に驚いたよ。
ハウリングブレード : 彼女の強さは鈍っていなかったから、
てっきり、症状が治まって闘士を再開したのかと……。
ヤーナ : なら教えてあげるよ。
アンタは恩義を理由に、不都合な真実から目を背けて、
大事な親友や恋人のことが見えなくなってるんだ。
ヤーナ : ユト姉はね、アンタの魂蝕症を治すために、
Sakilinaの魂を奪おうとしたんだ。
ヤーナ : それでしかアンタを救えないって思ってたから。
アルカディアから離れようとしないアンタと生きる方法が、
ほかになかったから……！
ハウリングブレード : なんてことだ……返す言葉もない……。
ハウリングブレード : そうか……ボクは怖かったのか。
闘士「ハウリングブレード」としての生き方が、
失われてしまうことが……。
ハウリングブレード : だから、オーナーの言葉を信じ続けていた親友のことも、
死ぬとわかっていながら止められなかった。
ハウリングブレード : すまない、ブルートボンバー……いや、ヘクトール。
ボクは、Sakilinaやユトロープが、
そうしたように、キミを救おうとすべきだった……。
ハウリングブレード : わかったよ。
ボクも、キミたちに協力すると約束しよう、
闘士を食い物にする、オーナーを止めるために……。
ヤーナ : ハウリングブレードが味方になってくれると、
こちらとしても、とても心強いよ。
ハウリングブレード : 「レザラ」と呼んでくれ。
ボクはもう、ハウリングブレードじゃないからね。
ヤーナ : レザラ、
ジムには姉さんもいるし、戻ってこれからの話をしよう。
レザラ : ああ。
そうだ、少しだけ寄り道したいところがあるから、
その後で、ジム・トライテールに向かわせてもらうよ。
レザラ : すまないんだが、ボクの家はペット禁止でさ……
どうか、この子たちを預かってやってくれないか？
ヤーナ : 喜んで引き取るよ。
妹も猫を飼いたがっていたし……
ヤーナ : ネユニは上でユト姉の看病をしてるのかな？
ちょっと見てくるよ。
レザラ : ……ボクも行っていいだろうか？
ヤーナ : もちろん。
姉さんと積もる話があるだろうからね。
ヤーナ : 大変だ！
ネユニとユト姉がいない！
ヤーナ : 疲弊した姉さんを外に連れ出すなんて考えられないし、
いつも持ち歩いている端末も置いたままだった。
きっと何かあったんだよ……！
低くかすれた声 : 生身の挑戦者、Sakilina。
私はアルカディアを盛り上げることに貢献してきた君を、
とても高く評価している。
レザラ : これは、オーナーの声だ……！
アルカディアのオーナー : しかし、私の筋書きを勝手に書き換えたことは、
とてもじゃないが容認できない。
アルカディアのオーナー : これ以上、好きにさせないため、
ユトロープとネユニの身柄は確保させてもらったよ。
アルカディアのオーナー : メテムを締め上げたことで、
君たちが魂蝕症の治療にやっきになっていることも承知だ。
アルカディアのオーナー : ああ、それから……
君たちが欲してやまない再生因子を強化するための薬、
通称「生命の雫」の在処も、私は知っている。
レザラ : いったい、何が望みですか？
アルカディアのオーナー : 簡単なことだよ。
これまでどおり、Sakilinaが、
アルカディアで闘い続けること。
アルカディアのオーナー : そして、私の筋書きに二度と手を加えないことだ。
統一王者になれば約束どおり、魂資源は好きにすればいいし、
なんなら「生命の雫」の在処を教えよう。
アルカディアのオーナー : 私からは以上だ。
アルカディア最高峰の闘いの舞台、
ヘビー級での活躍を楽しみにしているよ……。
レザラ : オーナーの方が、一枚上手だったようだ……。
下手に動くとユトロープたちが何をされるかわからない。
ヤーナ : もう、Sakilinaに、
統一王者になってもらうしかないってことか……。
ヤーナ : ありがとう。
これからも、セコンドとして精一杯サポートするよ。
レザラ : オーナーの暴挙はとても看過できない。
ボクもできることで協力させてもらうよ。
レザラ : 二度と、大事な人を奪われたくないからね……。
レザラ : 病床の身のユトロープと、幼い子どもをさらうなんて……
何としても、彼女たちを無事に取り戻そう。
レザラ : そして、闘士を食い物にするオーナーを止めるんだ。
ヘクトールもきっと、それを望むだろうからね。
ヤーナ : …………。
ヤーナ : ……ごめん、大丈夫だ。
こっちが下手な動きをしないかぎり、
ユト姉とネユニは、きっと無事でいてくれる……。
ヤーナ : アンタはアンタなりの方法で、
統一王者の座を勝ち取るための準備をしておいて。
ヤーナ : さらに激しい闘いに備えてトレーニングしたり、
ゆっくり身体を休めたり、とにかく次戦に備えてほしい。
ヤーナ : アンタに頼るしかなくて心苦しいけど、
ヘビー級の試合が始まる頃合いになったら、
またここに来て……頼んだよ！
ヤーナ : Sakilina、様子を見にきてくれたのか？
さらわれた、ネユニとユトロープのことは心配だけど、
下手に動くとオーナーに何をされるかわからない。
ヤーナ : でも、手をこまねいているわけにもいかないからね。
私たちにできることがないか、レザラと探っておくよ。
ヤーナ : アンタが次に挑むのは、いよいよ最高峰のヘビー級だ。
これまで以上に激しい闘いとなるから、
しっかり、トレーニングを積んでおいてくれ！
ガブロ : やあ、生身の挑戦者、Sakilina！
僕に声をかけてくれてありがとう。
ガブロ : さあ、「クルーザー級」の取材を始めよう。
ベテラン揃いの階級でも、目覚ましい快進撃を見せてくれたね。
ガブロ : ブルートアボミネーターとの場外戦では手に汗握り！
王者、ハウリングブレードの引退を賭けた大一番の興奮は、
いまだに覚めることがないよ……！
ガブロ : 今回も、客席や中継でじっくり観戦させてもらったが、
闘ったキミから生の声を聞かせてほしい。
さあ、お願いできるだろうか？
ガブロ : よしよし、フツフツと想像力が湧いてきたぞ。
僕もシュガーライオットに負けないくらい、
キミの闘いを漫画というアートで表現しよう！
ガブロ : こんなのはどうだ……
「ダンシング・グリーン」との熱狂のダンスバトルが、
さらに白熱したらどうなったのか？
ガブロ : 奔放な「シュガーライオット」のイメージが爆発し、
とんでもないものを具現化したら？
ガブロ : 「ブルートアボミネーター」による怒涛の攻撃が、
熾烈さを極めたらどんな地獄に？
ガブロ : もし、「ハウリングブレード」がノックアウトされず、
魔物の魂を暴走させてしまったら？
ガブロ : うんうん、「闘え！　生身の挑戦者　クルーザー級編」
これは類まれな傑作になりそうだ……！
ガブロ : 忙しい中、取材を受けてくれて感謝する。
次の階級も期待してるから、どうかがんばってくれ！
ガブロ : やあ、生身の挑戦者、Sakilina！
僕に声をかけてくれてありがとう。
ガブロ : 雑談したいっていうのかい？
そうだな、改めて自己紹介でもさせてもらおうか……。
ガブロ : 僕は絵で語られる物語、漫画というものを描いていてね。
主にアルカディアの闘技を題材にしているんだ。
ガブロ : 自分で言うのもなんだけど、
僕の作品はソリューション・ナインで大人気なんだよ。
ガブロ : 新たなスター闘士である、キミの闘いを漫画にしたら、
それこそ大ヒット間違いなしさ！
