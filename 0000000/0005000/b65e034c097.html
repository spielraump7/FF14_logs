サンクレッド : それにしても、クリスタルはザハラク戦陣か……。
面倒なところに運ばれたものだな。

この身を切り札として、必ずや勝利へと導きましょう。

ガ・ブみたいな子が増える前に、クリスタルを奪還するわよ。

動きがあるとすれば、蛮神「イフリート」の召喚が行われる、
砦の最深部……祭壇あたりか。
アルフィノ : Walt、戦いの準備はいいかい？
神降ろしが行われる前に、ザハラク戦陣に乗り込もう！
サンクレッド : ……妙だな。
祭壇近くだってのに、警備が薄すぎる。
アリゼー : どうやら、アマルジャ族の怠慢ってわけじゃないみたい。

相変わらず遅すぎだぜ、甘ッタレども。

しかし、なぜ……！
闇の戦士 : アシエンの野郎は、相も変わらず、
蛮神討伐によって蛮族どもを絶望させ、
「新たな神」を喚ばせようとしてるんだがな……。
闇の戦士 : 俺たちはやめたんだ、そういう面倒なやり方は。
なにせ、時間がないもんでな。

エオルゼア中を、一気に混乱に叩き落せる。
闇の戦士 : 第一世界のすべての命のために……
死んでもらうッ！
サンクレッド : おいおい、お前らにリベンジしたいのは、
あいつだけじゃないんだぜ！
アルフィノ : させるかッ！
カーバンクル・オブシダン！
闇の狩人 : あーあ、今度こそ仕留めてあげようと思ったのに。
でも、あっちの子はどうかな？

アリゼー : 問題……ないわ……。
絶対に、足手纏いにならないって……
約束したじゃない。

私、デキのいい兄と違って、頭より先に体が動くの。

お前が世界を引き裂くなら、
私は、それを斬り裂く剣になるッ！
闇の戦士 : 来いッ！
闇の戦士 : 邪魔する奴は、全員潰す……！
俺たちの世界のためにッ！
アルフィノ : させるものか！
私たちにも、守りたいものがある！
闇の戦士 : 軽すぎるんだよ……！
その一撃も、意志も、背負ってるモンも！
アリゼー : 立ち上がった……！？
これも、「超える力」の片鱗だというの！？
アリゼー : くっ、また……！？
どうにかして、一気にカタをつけないと……！
闇の戦士 : 遊びは終わりだ、甘ッタレども！
サンクレッド : クソッ……身動きが……！
闇の戦士 : さあ……
そろそろ終わりだぜ。
アルフィノ : これは……拘束が解けた……！？
闇の戦士 : ……どういうつもりだ。
俺たちまで裏切るのか？
怪しげな男 : 私なりの手法に沿うだけです……。
そもそも、裏切りとは、信じたものを破ればこそ……。
怪しげな男 : 第一世界に住まえる者の魂を、
この原初世界に送ることが、本当の救いであると……
あなた方は、心から信じておいでですか？
アルフィノ : その声……！
まさか、ウリエンジェなのか！
ウリエンジェ : 申し訳ありません……。
己が役目を果たすため、しばし、闇に紛れておりました……。
アリゼー : 馬鹿……。
ちょっとだけ、本気で裏切ったんじゃないかって、
心配したじゃない……。
アリゼー : 弁解は、あとでたっぷり聞かせてもらうわ。
今のあなたは味方……それでいいのよね？
ウリエンジェ : ええ、無論です。
ウリエンジェ : ……何なりとご命令を。
闇の戦士 : 数は互角、か。
上等だ……まとめて蹴散らしてやる。
闇の魔人 : ……了解した。
アリゼー : あいつら、大技を仕掛けてくるみたい。
だったら、ひとつ提案があるの。
アリゼー : あなたたちの力を、私に貸して。
あっちの技が発動する前に、みんなのエーテルを剣に集め、
光の刃として一気に放つのよ……！
アリゼー : 私ひとりじゃ、刃の威力も、構築するための時間も、
圧倒的に足りないけれど……
ひとりぼっちじゃないなら、勝てない賭けじゃないわ。
アリゼー : 行くわよ……ッ！
闇の戦士 : 互いに大技ってわけか……。
だが、勝つのは俺たちだッ！
アリゼー : 私に力を貸して！
みんなのエーテルを、私にッ！
アルフィノ : まずは、私がエーテルを付与する！
Waltは、敵を引き付けてくれ！
アルフィノ : ハァッ……私はこれが限界だ……！
Waltも、アリゼーにエーテルを！
サンクレッド : 邪魔者は俺が引き付ける！
アリゼー様のところに、早く！
アルフィノ : それ以上のエーテル付与は、君の負担になりすぎる。
一旦、退くんだ！
闇の戦士 : クソッ……！
ブランヘルツ、Waltを止めろ！
ウリエンジェ : 我が力を預けます……。
二度はない決戦の機、必ずや勝利を……！
ウリエンジェ : Waltさん……
アリゼー様に、再びエーテルを……！
闇の戦士 : いけるな、ジャ・ルマレ！
あの女の詠唱を止めるんだ！
アルフィノ : アリゼーが狙われている……！
Walt、あの狩人を止めてくれ！
闇の戦士 : 急げよ、ナイユベール！
お前の魔法で、一気に終わらせろ！
アルフィノ : 今だ、アリゼーに最後のエーテルを！
闇の戦士は、私が食い止める……ッ！
アルフィノ : 大きな力を感じる……！
決めるんだ、アリゼー！！
アリゼー : いくわよ……！
この剣で、はだかる者を叩き斬るッ！
闇の戦士 : チクショウ……俺たちは、こんなところで……！
アルフィノ : アリゼー！
大丈夫か……！？
アリゼー : からっからだけど……大丈夫……。
みんなのおかげでね。
サンクレッド : ……勝敗は決した。
お前たちに、こいつは殺せない。
闇の戦士 : さて、そいつはどうかな……。
俺たちも、まだ終わるわけにはいかないんでね……。

まさか、アシエンの不死と同じ……！

己の魂を「次元の狭間」に送り、死をまぬがれるように……
彼らは、光のクリスタルをもって、魂を越境させる……。
闇の戦士 : ハッ、ご丁寧な解説ありがとうよ……。
つまり俺たちも、アシエンと同じように不死になれる。
……この敗北は、終わりじゃない！
アルフィノ : 待ってくれ！
アルフィノ : その方法を使うには、元の世界で、
魂の器である肉体の死を迎えなければならないはず。
ならば、君たちは……！
闇の戦士 : ……そうだ。
俺たちは、俺たちの世界を救うため……自ら命を絶った。
闇の戦士 : 「光の戦士」なんて肩書を背負ってたからじゃねぇ。
最初はただ、旅の中で出会った奴に、
手を貸しただけだったんだ……。
闇の戦士 : そんな因縁がいつの間にか広まって、俺たちは英雄と呼ばれた。

世界は「光の氾濫」で無に帰るときやがった……ッ！
闇の戦士 : なんでだよ……。
そんな結末のために、俺たちは歩んできたんじゃない……！
闇の戦士 : なあ、お前ならわかるだろ……わかれよッ！
世界をあんな風にしちまった俺たちは……
俺たちだけは、何があっても諦めちゃならないんだ……！
闇の魔剣士 : そうだよなァ……アルバート……。
この程度の痛み、死んだときに比べたら、なんてことはねぇ。
闇の導師 : 何度でも、戦いましょう。
はるかなる故郷の仲間に……せめてもの救いが訪れるように。
ウリエンジェ : これほどの嘆きに、なお多くの犠牲を重ねるなど、
我が師であれば……いえ、私自身が認めない。
今、万事の糸を縒り、新たな道を紡ぎましょう……！
ウリエンジェ : あなたの、光のクリスタルを掲げてください！
さあ、早く……！
アルフィノ : ここは……？
サンクレッド : この声、まさか……！
ウリエンジェ : ハイデリンよ……！
ふたつの世界の光のクリスタルをもって、

ミンフィリアを、解き放ちなさい！

「光の氾濫」も、避けることができるはずです。

ハイデリンなど……
俺たちの祈りにも、応えてこなかったくせにッ！

ゾディアークがアシエンを介さなければ干渉できないように、
ハイデリンも、直接「光の氾濫」を止めることはできないの。

こうして使徒として切り離された……。

大丈夫……第一世界は、決して滅ぼさせない。
