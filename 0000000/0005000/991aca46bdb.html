懐郷の民

アルフィノ : さて……クリスタルの取引現場は、この周辺のはず。
あとは、「アラミゴ人」という漠然とした相手を、
どう絞っていくかだが……。
アルフィノ : 不穏な動きをしている者がいれば、
何かしらの噂になっているかもしれない。
このリトルアラミゴの住民たちから、話を聞いてみよう。
アルフィノ : 私とアリゼーは、集落の西側で聞き込みをしてみる。
Walt、君には東側を頼めるかい？
後ほど、またこの場所で合流だ。
シフリッド : 最近、変わったことはないか？
そんなの、もちろん大ありさ！
「鉄仮面」卿という、最高の革命家が登場したんだからね！
シフリッド : 彼のアラミゴ奪還にかける熱意には、心が震えたよ。
オレも解放軍の一員として、
ガレマール帝国の連中を、この手でなぎ倒すんだ！
テールボット : ……話を聞いて回ってる？
そうはいっても、今はみんな、
「鉄仮面」卿のことばかりじゃない？
テールボット : ホント、人気者だよ……。
名前のとおり、いつも仮面を被っていて、
その下の素顔は、側近にも明かしていないらしい。
テールボット : アラミゴ解放運動中につけられた醜い傷があるだとか、
帝国に追われている指名手配犯だとか……
みんなは好き勝手に噂をしてるよ。
オテリン : ああ……？
大した事件なんてねえよ。
俺たちアラミゴ難民は、いつもどおりの酷い暮らしさ。
オテリン : だが、噂の「鉄仮面」卿の誘いに乗れば、
諦めてた故国奪還の夢も、少しは現実に近づくのかもな……。
なんでも、今までの解放軍にはない秘策があるそうだぞ。
オテリン : クソッ、俺だって、もう少し回復してれば……。
このまま、この地で弱っていくくらいなら、
刺し違える覚悟で解放運動に加わっても……。
アリゼー : どうやら、おかしな奴が現れてるらしいわね……。
アルフィノ : Walt、おかえり。
君の方はどうだった？
アルフィノ : ……やはり、「鉄仮面」という人物の話題ばかりだったか。
私とアリゼーの方も、概ね同じ状況だったよ。
アルフィノ : アラミゴ解放運動を否定する気はないが、
突然活発に動き出した「鉄仮面」のことは、少々気がかりだ。
アルフィノ : アラミゴを占領しているガレマール帝国に対し、
強気に出れるだけの材料があるということだからね。
それが、蛮神召喚でなければいいのだが……。
アルフィノ : ここの集落の顔役から、
もう少し詳しい話を聞いてみようと思うんだ。
顔見知りの君に、仲介を頼めると助かるよ。
アリゼー : アラミゴがガレマール帝国に占拠されたのって、
確か20年くらい前だったわよね。
鉄仮面とやらは、今の今まで何をしてたのかしら？
アルフィノ : ただの懸念で済むといいが……。
グンドバルド : 冒険者よ……。
先ほどから、仲間の話を聞き回っているようだが、
また不穏な兆候でもあったのだろうか。
グンドバルド : ……なるほど。
お前たちの気にしている「鉄仮面」卿とは、
近頃、アラミゴ解放軍の中で頭角を現してきた人物だ。
グンドバルド : 素性は知れないが、解放運動に並々ならぬ熱意を持ち、
国外に逃れた同志たちを、急速にまとめはじめている。
外側からアラミゴを救うため、何か策を練っているらしい。
グンドバルド : アラミゴの民として、彼を疑いたくはないが……
以前、若人らの暴走をとめてくれたお前の目に、
曇りがないことも知っている……。
グンドバルド : よし、「カルン埋没寺院」に行くがいい。
そこで、鉄仮面卿みずからが、仲間を募る演説をしている。
お前自身の目で、見定めるのだ。
アルフィノ : ふむ……よいタイミングだったということかな。
さっそく、その演説を聞きに行ってみよう。
アリゼー : だったら、私はここで留守番してるわ。
サンクレッドとすれ違いになったら、困るだろうし。
……くれぐれも気をつけてね。
グンドバルド : 熱く燃える心が、人を動かし、希望の星となることがある。
しかし、その星が見せる道が、必ず正しいとは限らない……。
我々アラミゴの民は、幾度もそうした誤りを繰り返してきた。
グンドバルド : そして、それが渦中の者にこそ正せない誤りだと知っている。
だからこそ、お前の目にゆだねよう。
ウィルレッドもまた……それを望むであろうよ。
アリゼー : こちらも、情報収集を進めてみるわ。
……ザナラーンの地をひとりで散策するなんて、
どれくらいぶりかしら。
アルフィノ : よし……。
演説は、中で行われているようだ。
我々も入ってみよう。
鉄仮面 : 知ってのとおり……
20年前、我らが故郷「アラミゴ」は、
鉄の侵略者「ガレマール帝国」に占拠された。
鉄仮面 : きっかけは、長く圧政でアラミゴを苦しめていた、
暴君テオドリックを排するための内乱……
その直後の混乱を狙っての、非情な蹂躙だった。
鉄仮面 : しかし、君も……ここにいる誰もが、
かつてアラミゴが領したギラバニアの地を、
残してきた同胞を、忘れた日はないだろう。
鉄仮面 : かの地の惨状を、想像してほしい。
雄大な山々の間に、同胞の命を捧げて築かれた鉄の砦が並び、
豪然とした大地は、乾くままに同胞の血を吸っている。
鉄仮面 : 残された国民のうち、力と誇りを持つ者は惨たらしく殺された。
力のみを持つ者は、武器を握らされ、
帝国の雑兵として異郷に送られて、そこで死ぬ。
鉄仮面 : 力すらも持たざる者は、みじめにも侵略者に道の中央を譲り、
あらゆる尊厳を奪われながら生きている……ッ！
鉄仮面 : 憎き総督、ガイウス・ヴァン・バエサル亡きあと、
件の「新総督」が赴任して以来、弾圧は加速している。
鉄仮面 : もはや、エオルゼア諸国の助けを待つ時間はない。
奴らは口約束を重ねるが、利のない戦いと踏んで、
決して動きはしないのだ！
鉄仮面 : 地獄と成り果てた故郷で、今なお再起を待つ友のため、
我々はすぐに立ち上がらねばならない。
今度こそ、革命を成すという覚悟を持って！
鉄仮面 : さあ、私とともに来る者は声を上げよ！
その誇りにふさわしい力を与える準備が、我々にはある。
ギラバニアの山並に、故国の旗を再び仰ぎ見ようではないか！
アルフィノ : 誇りにふさわしい力、か……。
いよいよ疑念がぬぐえないな。
アルフィノ : ……おや？
あのふたりは…………
イダ : ウソでしょ……！
ふたりとも、どうしてここに……！？
アルフィノ : イダ、パパリモ！？
無事だったのか……！
パパリモ : 驚いた、まさかこんなところで会うなんてね。
パパリモ : ……再会にひたるには、場所が悪い。
とりあえず、ここを離れよう。
パパリモ : なんだか、一層たくましくなったんじゃないか？
お互いに無事で再会できて、何よりだよ。
イダ : まさか、Spielraumたちと、
こんなところで会えるなんて！
アルフィノ : イダ、パパリモ……！
無事でいてくれて、本当によかった……！
イダ : アルフィノ様と、Spielraumもね！
みんななら、ぜーったい無事だって信じてたけどさ！
パパリモ : 「暁」の指名手配がとけて、
クリスタルブレイブの解散が行われたとは聞いてたからね。
君たちが尽力してくれたんだって、思ってたよ。
アルフィノ : 知っていたのか……！
しかし、それならばどうして今まで……？
パパリモ : 僕たちは、あの戦勝祝賀会から逃げたあと、
古いツテを頼って、アラミゴ解放軍に身を寄せていたんだ。
イダ : そんで、匿ってもらった恩を返すため、
彼らの活動に力を貸してたってワケなんだけど……
もしかして、知らなかった？
パパリモ : 「暁」の嫌疑が晴れたと聞いたときに、
僕らの状況をしたためた手紙を、君たち宛に出したんだ。
リンクパールは、逃亡中に壊しちゃってたからね。
パパリモ : でも、君たちの反応を見るかぎり、
手紙は、そちらに届いていないようだ。
イダ : 信頼できる子に託したと思ったんだけどなぁ……。
解放軍内の検閲で引っかかっちゃったんだね、きっと。
「あの人」は、「暁」とかに介入されるのを嫌がってるし。
アルフィノ : 「あの人」……？
まさか、鉄仮面卿のことか？
アルフィノ : そんな……。
彼はいったい、どういう人物なんだ？
さきほどの演説は、いかにも熱がはいっていたが……。
パパリモ : ああ、演説をしてたのは、残念ながら影武者だよ。
僕らは、解放軍の一員として協力を要請されたときに、
本物とされる人物と会っているが……あれとは別人だ。
パパリモ : 本物の鉄仮面はとても用心深くて、
多くの人の前に姿を晒すようなことはしないのさ。
どのみち仮面をつけているし、気づく人はまずいないからね。
イダ : 命を預ける仲間にまで、素性を隠し続けるなんて、
なーんかイヤな感じじゃない？
イダ : い、いや、アタシの仮面は、別問題だとしてさ！？
イダ : でも、鉄仮面って、解放軍の中ですごーく人気があるんだ。
アタシたちは、仲間をこのまま鉄仮面に任せていいのか、
見定めるために演説を聞きに来たってわけ。
パパリモ : 結果は……
本人が出てきていない時点で、お察しだけどね。
パパリモ : ……それで、君たちの方は？
どうして鉄仮面の演説に？
イダ : ええっ！？
蛮神召喚のためのクリスタルを、
アイツが受け取ったかもしれないの！？
パパリモ : アラミゴ解放軍にそんな計画はなかったけど……
鉄仮面なら、独断でどうにでもできるか。
パパリモ : 彼が匂わせている切り札がなんなのか、
僕らも前々から気になっていたんだ。
……蛮神召喚も、ない話ではないかもしれない。
イダ : その調査、アタシたちにも協力させて！
鉄仮面が悪巧みしてるとしたら、絶対に暴いてやるんだから！
パパリモ : ああ、そうだとも。
「暁の血盟」の活動に、僕らも復帰させてくれ！
イダ : 鉄仮面のヤツ……
本当に、アラミゴ奪還のために蛮神を使おうとしてるなら、
絶対に許せないよ！
アルフィノ : 思わぬ妨害もあったようだが、ふたりと再会できてよかった。
タタルにも、早急に連絡をしておこう。
まずは力をあわせて、この件を解決しなくては！
