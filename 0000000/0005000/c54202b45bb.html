戦う者たちの休息

フォルタン家の騎兵 : おや、Walt様！
ちょうど良いところに来てくださいました。
フォルタン家の騎兵 : 実は今しがた、ボーレル家の使いの方がいらしたのです。
貴方様宛に、ボーレル子爵……
つまり、アイメリク卿からの伝言を、預かっておりますよ。
フォルタン家の騎兵 : なんでも、貴方様を自宅に招いて、
会食の席を設けたいのだとか。
フォルタン家の騎兵 : お忘れになっているかもしれませんが、
使いの方いわく、ファルコンネストでの式典に貴方様を招く際、
アイメリク卿が、そのような申し出をされたようで……。
フォルタン家の騎兵 : あの式典以後、あまりに多くのことがありましたが、
平和に祝杯をあげるとしたら、確かに今が最適でしょう。
……それで、いかがいたしますか？
フォルタン家の騎兵 : えっ！？　わ、私にそれを聞きます！？
そうですね……貴方様とアイメリク卿の仲を思えば、
何を着ていっても、問題はないかと存じますが……。
フォルタン家の騎兵 : 困ったら、とりあえずイシュガルディアンコートを羽織れ！
……と、以前エマネラン様から伺ったことがあります。
フォルタン家の騎兵 : ま、まあ、それに限らず……
お支度を整えられたら、聖ガンリオル占星院のすぐ近くにある、
「ボーレル子爵邸」に向かってくださいませ。
フォルタン家の騎兵 : よろしければ、聖ガンリオル占星院のすぐ近くにある、
「ボーレル子爵邸」へ。
ボーレル家の家令 : Walt Spielraum様ですね。
ようこそ、お越しくださいました。
ボーレル家の家令 : わたくしは、このボーレル家で家令を務める者にございます。
……とはいっても、使用人は片手で数えるほどですが。
ボーレル家の家令 : 此度は、我が主アイメリク様たっての希望で、
ささやかながら、お食事の席を設けさせていただきました。
間もなく支度も整いますので、どうぞ中でお待ちください。
ボーレル家の家令 : 貴方様も、我が主も、立場は違えど戦を続けてきたお方。
今日だけは心身をお休めになって、
思い出話にでも、花を咲かせてくださいませ……。
アイメリク : ……それで、雲海のモーグリ族にこき使われたのか！
なるほど、あのラッパは、君の苦労の結晶だったわけだ。
アイメリク : ふふ、君の旅の話は飽きないな。
報告として聞いたときとは違う発見がある。
アイメリク : ずっとこの地で生きてきたというのに、
君の目を通して見ると、まるで別世界のようだ。
……いいものだな、旅というのは。
アイメリク : まさか、誘ってくれるのか……！？
実は私も、先にドラヴァニア雲海で見た風景を思い出しては、
まだ見ぬ地を目指してみたいものだと思っていてな？
アイメリク : 喜んで、君の誘いを受けたい……
と思うのだが、今はまだ議長を引き受けたばかりの身。
それは、いつかの夢にとっておくことにしよう。
アイメリク : ……君に助けられて、今やっと、
この国は真の変革を迎えることができた。
アイメリク : だが、共和制への移行ですら、変革のたった一歩に過ぎない。
これからのイシュガルドは、制度の面のみならず、
民ひとりひとりが変わらねばならない時期を迎えるはずだ。
アイメリク : 私の生きているうちに、果たして安定するものか……
先が見えない道だが、戦いに散った者のためにも進んでいこう。
アイメリク : 何より、この目には今も、
邪竜に向かって進む君の背中が、しかと焼き付いているのだ。
それを追わずに立ち止まることなど、できるはずがないだろう？
アイメリク : 今、新設された議会では、
他国とどのように協力関係を築くべきか、
議論が進められている。
アイメリク : 無論、当面は国内の平定が最優先ではあるが、
先の合同演習についての恩もある……
できることから、協力をしていきたい。
アイメリク : 他ならぬ君にもだ。
君に報いることが、「暁」だけでなく、
三国、ひいてはエオルゼアへの恩返しになるだろう。
アイメリク : この地の民の誰もが、そう信じて疑わないほどのことを、
君はやりとげてくれたということだ。
本当に……ありがとう。
アイメリク : ……ところで。
これは、議長としてではなく、ごく個人的な興味なのだが。
アイメリク : 君は、これからどうするつもりだ？
「暁」としての目的というより……
君自身の想いを聞かせてくれないか。
フォルタン家の騎兵 : お食事中、失礼いたします！
アイメリク : フォルタン家の……？
何かあったのか？
フォルタン家の騎兵 : ハッ！
そちらの英雄殿に、至急お伝えすることがあり、
無礼を承知で案内していただきました。
フォルタン家の騎兵 : 先ほど、サンクレッド様が、負傷したお仲間を連れ、
フォルタン伯爵邸に駆け込んでいらっしゃったのです。
フォルタン家の騎兵 : アルフィノ様とタタル様が手当てにあたっておいでですが、
あまり状態がよろしくないらしく……
至急、いらしてほしいとのことです！
アイメリク : 私も行こう。
手を貸せることがあるかもしれない。
フォルタン家の門衛 : お帰りなさいませ、お客人。
「フォルタン伯爵邸」に入られますか？
アイメリク : これは、いったい……。
サンクレッド : 彼女は、アリゼー……
アルフィノ様の、双子の妹君さ。
サンクレッド : ……やられたよ、闇の戦士たちだ。
サンクレッド : ファルコンネストでの式典のあと、
連中の足取りを追っていたんだが……
黒衣森で、闇の戦士に狙われたアリゼー様を助けてな。
サンクレッド : 彼女も、別の線から連中に疑念を抱いて、
独自に追跡を続けていたらしい。
サンクレッド : 一時は、追っ手を完全に振り切れたと思ったんだが、
クルザスに入る手前で、奴らの一味に強襲された。
アリゼー様は、交戦中に毒矢を受けてな……。
アルフィノ : 急に呼び立てて申し訳ない。
アイメリク卿も、来ていただけて助かります。
アイメリク : こちらは構わない。
しかし、妹君の容態は……？
アルフィノ : ……応急処置は施しました。
今すぐに命を落とすようなことはないでしょう。
ですが、症状が治まらず、衰弱しきっている……。
アルフィノ : 妹……アリゼーは、私と同じ……
祖父ルイゾワが命をかけて守ったエオルゼアを、
見定め、救うためにやってきたのです。
アルフィノ : しかし、この地で見たのは、
人同士のいさかいや、欺瞞と綺麗ごとばかりにも思える政治……
彼女はひどく怒り、悩んでいました。
アルフィノ : そして、そういった問題に関わろうとする、
私や「暁」と距離を置いた……。
アルフィノ : 彼女は、彼女なりのやり方で、
エオルゼアの明日を模索するといっていました。
きっと今回も、そのために闇の戦士を……。
アルフィノ : アイメリク卿、彼女は「暁」の一員ではありませんが、
エオルゼアを救いたいという思いをともにする仲間。
何より……私の大事な妹なのです。
アルフィノ : どうか、彼女の治療に、
ご助力いただけないでしょうか……！？
アイメリク : 断る理由があるものか。
治癒師を手配し、最善を尽くさせよう。
アイメリク : 彼女を、神殿騎士団病院へ。
院長のエーベル殿に事情を説明し、治療にあたってくれ。
責任は私が負うと伝えてくれていい。
アリゼー : 待って…………。
あなたたちに……話さなきゃいけないことが……。
アリゼー : 闇の戦士は、アシエンと結託してる……。
蛮神を討伐してまわっているのも、企みの一環よ……。
アリゼー : 次は、「ゼルファトル」の地で、
蛮神「ガルーダ」を再召喚させると言っていた……。
倒すために、わざとね……。
アリゼー : どうか、蛮神「ガルーダ」の再召喚を……
奴らの企みを、阻止、して……！
タタル : あ、あのっ、あのっ！
私、ほかの賢人のみなさんにも、連絡してくるでっす……！
アイメリク : サンクレッド殿、
病室まで、アリゼー殿に付き添っていただけるだろうか？
毒を特定するために、証言をお願いしたい。
サンクレッド : ああ、了解した。
アイメリク : お互い、どうにも落ち着けない宿命にあるようだ。
食後の紅茶は、日を改めて楽しむとして……
今は、対策を考えねばな。
エドモン・ド・フォルタン : たとえ普段は離れていようとも、
兄弟というのは、不思議な繋がりを持つもの……。
アルフィノ殿も、さぞや妹君を案じていることだろう。
ソレット : あっ、Walt様……。
わ、私……新人のソレットと申しますっ……。
ソレット : ご、ごめんなさい……。
憧れの英雄様に声をかけられるだなんて……
思っていなかったので……その……。
フォルタン家の家令 : お帰りなさいませ、Spielraum様。
おや、未だにこのように言われるのが意外ですかな？
フォルタン家の家令 : 伯爵位をお継ぎになったアルトアレール様は、
まず最初に、我ら使用人にこう申したのですよ。
「一族の盟友」は、常に家族同然に迎えるべし、と。
アルフィノ : エスティニアン殿を救った治療師たちなら、
必ずや、アリゼーを救ってくれるはず。
……今は、信じて彼らに任せよう。
アルフィノ : それに、ベッドの脇で祈るばかりでは、
またあの人に笑われてしまう。
私は、自分のすべきことをするんだ……！
アルフィノ : Walt、君もどうか力を貸してほしい。
