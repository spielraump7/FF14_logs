ルヴェユールの双子

アルフィノ : すまない、ここで考えに耽るべきではないな。
フォルタン家への報告は騎兵たちに頼むとして、
我々は、アイメリク卿のもとに向かわないかい？
アルフィノ : 事の顛末を、彼も気にかけているだろうし……
どうしても、アリゼーの容体が気になるんだ。
アルフィノ : もちろん、休息をとりながらでいい。
「神殿騎士団本部」に行ってみよう。
アイメリク : Walt殿、帰還したのか！
では、作戦は無事に……？
アイメリク : ……そうか、よかった。
蛮神「ガルーダ」の再召喚を阻止してくれたこと、
イシュガルドの一員としても、深く感謝しよう。
アイメリク : こちらではその間に、
タタル嬢の要請を受けた、ヤ・シュトラ殿が面会にきた。
今は、皆でアリゼー殿を看ているよ。
アイメリク : 治療の合間に、君たちが以前、
闇の戦士と相対したときの話を聞かせてもらった。
残念ながら、それを活かした情報収集には至っていないが……。
アイメリク : 何にせよ、まずはアリゼー殿の病室に行こう。
……大丈夫、彼女はとても強かった。
さすがはアルフィノ殿の妹君、といったところかな。
アルフィノ : アリゼー！
もう起きられるのかい？
アリゼー : ええ、おかげさまで。
……迷惑をかけてしまって、ごめんなさい。
アリゼー : あなたも……
先陣を切って、ゼルファトルに乗り込んでくれたと聞いたわ。
本当にありがとう……。
ヤ・シュトラ : 顛末は聞いてるわ。
そちらは、再召喚を止めることができたの……？
サンクレッド : ふむ……。
闇の戦士たちが、アシエンと密会しているところは、
俺もアリゼー様とともに目撃している。
サンクレッド : しかし、別の世界を救うために動いていたとは……
そんな話、信じられるのか？
アルフィノ : 「逆さの塔」で、星の代弁者として、
ミンフィリアが語ったこととも、相違はないように思える。
……まったくの嘘ではないだろう。
ヤ・シュトラ : アリゼー様は、どうして闇の戦士の追跡を？
アリゼー : 私は、私なりにエオルゼアを知るため、各地を旅していたの。
「暁」や英雄の話は、風の噂でよく聞いたわ……。
アリゼー : けれど最近、別の気になる噂を耳にしてね。
イシュガルド動乱の裏で、各地の蛮神を倒している、
別の英雄たちがいると……。
アルフィノ : それが、闇の戦士たちだったということか……。
そして噂を追っていった君は、
闇の戦士とアシエンの関係を知った。
アリゼー : ……ええ、そのとおりよ。
アイメリク : しかし、彼らが霊災を引き起こすために動いているとして、
蛮神を召喚させ、あえて倒すという行為に、どんな意味が……？
アリゼー : 今、蛮神を続けざまに倒された各地の蛮族たちは、
警戒を強め、より強い状態で蛮神を召喚しようとしてるわ。
アリゼー : イシュガルドがニーズヘッグを討ったことも、
彼らにとっては、人への恐怖を煽る事実にすぎない……。
みんな、自分たちを守るために必死になってるのよ。
アイメリク : 力が力を渇望させる、恐怖と憎悪の連鎖……
「竜詩戦争」と同じだな。
アリゼー : そう、それこそが連中の狙い。
アシエンが不安を焚きつけて蛮神召喚を促す一方で、
闇の戦士が、それを倒してまわる……。
アリゼー : 力への渇望が連鎖して、
いずれ既存の神では太刀打ちできないと悟れば、
「新たな神」をも喚ぶだろうと、アシエンは言っていたわ。
ヤ・シュトラ : ……見過ごせないわね。
サンクレッド : 俺とアリゼー様の集めた情報によると、
蛮神召喚に必要なクリスタルは、
アシエンに誑かされた人間の手で、召喚者に提供されている。
サンクレッド : その経路を、暴いて叩けば……
加速する負の連鎖を、一時的にせよ止められる。
ヤ・シュトラ : それなら、私はクルルと一緒に、
ゼルファトルの追加調査に行ってくるわ。
私たちの力で追えば、新たな事実も見つかるかもしれない。
アイメリク : こちらも、最近召喚が行われたという、
グナースの塚付近を調査し、流通を洗おう。
アルフィノ : よいのですか、アイメリク卿……！？
アイメリク : アシエンとは、父の件で浅からぬ因縁があるからな。
……何より、今や我が国はエオルゼア同盟軍の一員なのだ。
この地に降りかかる脅威と、ともに戦う責務がある。
アイメリク : ただ……我々は、蛮神召喚に関する知識に乏しい。
可能であれば、賢人殿の知恵を借りたいのだが……。
サンクレッド : だったら、俺が付こう。
……よろしくお願いしますよ、アイメリク卿。
アルフィノ : 私たちも行こう。
蛮神召喚に関する動向を追っているウリエンジェを訪ねれば、
別の糸口もつかめるかもしれない。
アリゼー : 待って。
……私も連れていって。
もう体は動くし、絶対に足手纏いにはならないから。
アルフィノ : アリゼー、君は…………。
アルフィノ : ……いや、ともに行こう。
それが、君の見つけた道ならば。
タタル : そのお言葉を、お待ちしておりまっした。
アリゼー様、お着替えの準備はバッチリでっす！
アルフィノ : あまり大きな声では言えないが……
君とアリゼーは、蛮神「バハムート」の調査を進めていたね。
アルフィノ : 家の者から受けた報告によると、
彼女はその調査の合間に、行き先も告げず、
エオルゼアのあちこちを放浪していたようなんだ。
アルフィノ : 恐らく、彼女なりの道を模索し続けていたのだろう。
……その道が、まさかここで交わるとは思いもよらなかったよ。
アリゼー : ごめんなさい、待たせたわね。
アリゼー : おかげさまで、こっちは準備バッチリよ。
いつでも出発できるわ。
アルフィノ : ふむ、さすがはタタル……。
これなら、君と見間違えられることもなくなりそうだ。
アリゼー : ええ、まったく大助かりよ。
お父様とお母様ときたら、何歳になっても、
私たちに似た服ばかり寄越すんだから。
アリゼー : そのせいもあって、在学中は大変だったわ。
アルフィノを慕う子が、私を本人だと勘違いしたときなんか……
アルフィノ : ま、待ってくれ……！
その話をWaltの前ですることはないだろう。
アリゼー : 改めて、今回の任務に同行させてもらうわ。
よろしくね、Walt。
アリゼー : 第七霊災の真実を求める旅も、まだ終わってはいないけれど、
こちらの都合を顧みずにあなたを借りていくほど、
私も考えなしではないつもりよ。
アリゼー : あなたと、かの大迷宮に挑むためにも、
今は闇の戦士の問題解決に、全力で協力させてもらうわ。
アルフィノ : そういえば……
ウルダハの「戦勝祝賀会」から逃げるためのキャリッジは、
君が手配してくれたのかい？
アルフィノ : 確か、ルヴェユール家の子女から頼まれたと、
同乗した商人が言っていたと思うのだが……。
アリゼー : ああ、偶然、きな臭いことになってるって知ったからね。
知っていて見過ごすのもなんだし。
……今回は私が助けてもらったから、貸し借りはチャラよ。
アルフィノ : ……やはり君は変わったよ、アリゼー。
その様子だと、君の中で、
エオルゼアは守るに足るものだと判断されたのかな。
アリゼー : 私の力でエオルゼアを守ることができると思えるほど、
もう身の程知らずじゃないわ。
戦う意味にも、絶対の真理が見いだせたわけじゃない。
アリゼー : ただ、私は、今日を後悔しないためにここに来た。
今はそれがすべてよ。
アルフィノ : ……十分な返事だよ。
それでは、「砂の家」へと出発することにしよう。
ウリエンジェから、蛮神召喚についての動向を聞くんだ。
アルフィノ : 邪魔をするよ、ウリエンジェ。
ウリエンジェ : これは……。
知の神サリャクも預かり知らぬ……数奇な来訪者……。
私に、何か御用でしょうか……？
アルフィノ : 以前、闇の戦士に出会ったことについては、
君も報告を受けているだろう？
それに関して、新たな動きがあったんだ。
ウリエンジェ : 事情は承知いたしました……。
確かに、ここのところいくつかの蛮神が再召喚され、
何者かに討伐されているのは、私も知るところ……。
ウリエンジェ : そして、蛮族たちの神降ろしへの熱が、
以前より増しているのも、事実でしょう……。
アルフィノ : 我々のすべきことは、ふたつ……。
神降ろしに使われるクリスタルの供給を絶つことと、
それまで蛮神召喚を食い止めることだ。
アルフィノ : 闇の戦士とアシエンも、恐らく次の計画を進めているはず。
ウリエンジェ、何か手がかりになる情報を知らないかい？
ウリエンジェ : 運命は……手繰る者の手中に舞い込まん……。
まさに今しがた、神降ろしの兆候があるとの報告を、
受けたところです……。
ウリエンジェ : 外地ラノシアにて、
蛮神「タイタン」の信徒たるコボルド族が、
活動を盛んにしているとのこと……。
ウリエンジェ : 先の話に添うのであれば……
アシエンの後押しを受けた可能性が否めない……。
……向かってみては、いかがでしょうか？
アルフィノ : それは……確かに調査の必要がありそうだ。
あの辺りだと、キャンプ・オーバールックの黒渦団に、
詳しい事情を聞いてみるのが早いか……。
アルフィノ : 情報に感謝するよ。
引き続き、君も調査にあたってもらえるかい？
ウリエンジェ : ええ、もちろんです……。
アリゼー : ……ところでウリエンジェ。
あなた、私たちに何か隠していることはない？
アルフィノ : アリゼー？
それは、どういう……。
アリゼー : 単に、ウリエンジェにしては話が簡潔だってだけよ。
特に言うべきことがないなら、それでいいわ。
アリゼー : 私からも、ひとつ調べ物を頼んでいい？
闇の戦士が言っていたという、
「世界の統合」について、具体的なことが知りたいの。
アリゼー : あなたなら、調べる当てがあるんじゃない？
たとえば、そう……偽典「ゲルンの託宣」とかね。
ウリエンジェ : …………承知いたしました。
アリゼー : それじゃ、行きましょ。
キャンプ・オーバールックだったわよね。
ウリエンジェ : 調査は、よしなに進めておきましょう……。
アリゼー様とアルフィノ様のこと、
よろしくお願いいたします……。
ブルーエイディン少甲佐 : チッ、このまま事が大きくなったら、兵力がたりんぞ……。
軍令部に増援要請をしておくか……？
ブルーエイディン少甲佐 : んおっ、何の用だ？
生憎だが立て込んでてな、要件なら後にしてくれ。
ブルーエイディン少甲佐 : なに、「暁の血盟」だと……？
そうか、コボルド族の動向を知って駆け付けてくれたんだな！
がっはっは、そいつは百人力だぜ！
ブルーエイディン少甲佐 : まさに今、連中のことで問題発生中だ。
頼む、お前も手を貸してくれ！
