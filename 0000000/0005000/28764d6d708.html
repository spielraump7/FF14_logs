珍客来る

アルフィノ : ともかく、オメガ確保班は、
「グリダニア・ランディング」に集合することになった。
アルフィノ : アリゼーも先に向かっている。
私たちも「シド」たちと合流しよう。
アルフィノ : おや、ネロの姿が見えないようだが……。
シド : よう、来たな。
他の連中は、エンタープライズを見に発着場に行ってるぜ。
ネロの奴が、どうしても見せろとうるさいもんでな……。
アルフィノ : 大丈夫なのかい？
協力を申し出て来た相手とはいえ、ネロは元帝国軍人……
あまり自由にさせ過ぎるのも危険に思えるが……。
シド : わかっているさ。
お嬢さん方がふたりで監視してくれている。
っと、噂をすれば戻ってきたようだぞ……。
シド : よし、これで全員そろったわけだが……
このまま「カルテノー平原」に直行するか？
アルフィノ : いや、少しだけ時間をくれないか？
東部森林で仲間が繭の観測を行っているんだ。
アルフィノ : まずは彼らから、最新の状況を確認しておきたい。
その上で、改めて「オメガ」の回収に向かう人員を、
選出したいと思うのだ。
ネロ : ちんたらやってる場合じゃねぇと思うがな？
アリゼー : うっさいわね。
新入りは黙ってもらえる？
ネロ : へいへい……。
シド : よし、それじゃあ、
俺は「エンタープライズ」をモードゥナに回しておく。
「石の家」で改めて合流しよう。
アルフィノ : すまない、手数をかけるね。
観測を続けているヤ・シュトラやイダたちには、
私から報告しておく。
アルフィノ : Waltとアリゼーは、
先に「石の家」に戻っておいてくれ。
想定外の客人を招くことになるからね……。
シド : よし、モードゥナに直行する連中は乗せてってやる。
ランディングの係員、「ヒダ」に話しかけて、
エンタープライズに搭乗してくれや。
ヒダ : シド様のお連れ様ですね？
登録船名は……「エンタープライズ・エクセルシオ」。
それでは、ご案内致します。
ヒダ : シド様のお連れ様ですね？
登録船名は……「エンタープライズ・エクセルシオ」。
それでは、ご案内致します。
ネロ : ガーロンドのフネ、気密キャビンも備えてねぇなんて、
まったく最悪な乗り心地だったぜ……。
それにしても、妙な装置を積んでやがンな……。
ユウギリ : あの男……好かんな……。
アリゼー : 帝国出身ってだけで、差別する気はないけど、
ネロってのが油断ならないヤツってのは、わかる気がするわ。
シド : さすがにレヴナンツトールの街中に、
着陸させるわけにもいかんからな……。
ここに降ろさせてもらったぜ。
シド : さて、さっそくだが「石の家」に向かうとするか！
タタル : ここは、ネロという人にもお茶を出すべきか、
帝国軍と因縁がある「暁」の受付としては、迷ってしまいまっす。
ユウギリ : あの男が妙な気をおこさぬよう見張っておくさ。
アリゼー : バハムートを捕らえし者「オメガ」か……。
相対するのは、バハムートを超える存在として、
呼び降ろされた「新たな神」……なんだか因縁を感じるわね。
シド : 蛮神に対抗するため、古代アラグの兵器を用いる。
ガイウスも同じ事をやったが、ヤツはその武器を人にも向けた。
そうならないよう、俺たちが手綱を握るしかないな……。
フ・ラミン : ホーリー・ボルダーさんたちなら、
蛮族がらみの調査任務で、出払っているわよ。
フ・ラミン : 黒衣森の「光の繭」に関して、
賢人のみんなが対応に追われている状況だからこそ、
通常の任務を肩代わりしたいんだって、はりきっていたわ。
ホウメイ : 各々が使命に燃えておったわい。
志ある若者の眼ほど、まぶしいものはないのう。
ネロ : ここが、お前ら「暁」の本部か……。
賢人どもが集まる場所と聞いて期待していたンだが、
魔導技術の欠片もありゃしねぇ……。
ネロ : つくづくエオルゼアの奴らは、
遅れてるってのを実感するぜ……。
タタル : 失礼な人でっす。
この「石の家」にだって、最新の魔導技術があるでっす。
タタル : ウェッジさんが、私のために作ってくれた、
魔導湯沸かし器「シュワシュワケトルXIV世」という、
優れモノがあるのでっす！
アルフィノ : 待たせたね……。
アルフィノ : ヤ・シュトラやクルルさん、
それにサンクレッドたちは黒衣森に残って、
引き続き「光の繭」の監視と計測を続けてもらっている。
イダ : アタシは、付いて来ちゃった……。
パパリモが命を賭けて稼いでくれた時間を、
無駄にするわけには、いかないもんね……。
東方風の大男 : たのもおおおおおおおおお！
アリゼー : なにあの暑苦しそうなの……。
東方風の大男 : こちらが「暁」の賢人たちが住まうという「石の家」でござるか？
ウリエンジェ殿の紹介にて、訪ねて参った！
東方風の大男 : ぬっ、ぬぬぬぬぬ！
そこなる女人……ユウギリではござらんか！
探しておったぞおおおお！
アリゼー : ……まさか、あのむさ苦しいの、ユウギリの知り合い？
ユウギリ : ゴウセツ……！？
なぜ、お主がこのようなところに……。
ゴウセツ : ユウギリ、お主を探して、
はるばるドマより、船を漕いで参ったのだ！
ゴウセツ : どうにか港に辿り着いたまでは良かったが、
腹が減って、どうにも動けぬありさまでな……。
ウリエンジェ殿のご厚意がなければ、危ういところであった。
ゴウセツ : そして、飯喰いがてらに話を聞いてみれば、
ドマの民が、この地の開拓に参加しているというではないか！
しかも着いて早々、ユウギリと再会できるとは何たる幸運！
ゴウセツ : さあ、ユウギリよ、旅支度を整えよ！
我らが主君が、かつてない危機の火中にあるのだ！
急ぎ我らが故郷、ドマへと帰還し、馳せ参じねばならん！
ユウギリ : すまぬが、ゴウセツ……。
私には、請け負った任務がある故、すぐには戻れぬ。
ゴウセツ : なんと！　帰れぬと申すか？
貴様、大恩を忘れたわけではあるまいな！
一宿一飯の恩義があるやもしれぬが、今は主君の危機ぞ！
ユウギリ : 民を逃がし、何事があっても守り抜けと命じたのも我が主。
そして、ここにいる「暁」の方々は、我らを救ってくれた恩人！
その恩を返さず帰ったとあらば、主は私を斬り捨てるだろう。
ゴウセツ : ぐぬぬぬぬぬ……。
ユウギリ : そして今、守るべき民の安全を脅かす事態が起きている。
ゆえに任を引き受け、これより向かおうとしているところ。
任も民も捨て置いては、何より主君に面目が立たぬ。
ネロ : おいおいおいおい、何をグダグダやってるンだ？
「オメガ」の力は、帝国軍も狙ってるンだぞ。
こんなところで……。
ゴウセツ : て、い、こ、く、だとおおおおおお！
ゴウセツ : ユウギリよ、敵は憎きガレマール帝国の連中なのだな？
ゴウセツ : 敵が帝国の不届き者となれば、話は別だ！　早く申せ！
我が刃の味を帝国兵に味わわせる好機、逃してなるものか！
このゴウセツも助太刀するゆえ、さあさあ、戦場へ案内されよ！
タタル : Waltさん、お待ちくださっい！
受け取ってほしいものがあるでっす！
タタル : 覚えていますっか？
以前、ダイヤモンドフォージで、
お身体の隅々まで採寸させていただいた時のことを……。
タタル : 実は、Waltさんのために、
特別な旅装束を用意しようとしていたのでっす！
つい先ほど、完成したので、ぜひ受け取ってくださっい！
タタル : さぁさぁ、さっそく試着するでっす！
タタル : しっかり採寸したから、サイズはバッチリのはずでっすが……
ど、どうでっすか？
タタル : よかったでっす。
ほかならぬWaltさんに合う衣装……
製作に挑むために、これまで修行を続けてきたのでっす。
タタル : ブラッドパールのお守りも縫い込んでありまっす……。
だから、どうかご無事で、帰ってきてくださいでっす。
タタル : 絶対、あのネロという人には、
衣装を作って、あげないのでっす……。
ゴウセツ : 帝国軍の雑兵など一刀両断！
我が大刀で斬り伏せてくれるわ！
ユウギリ : まさかエオルゼアの地で、
ゴウセツと再会することになろうとは……。
アリゼー : あれが東方風の装束なのかしら……。
シド : どうしてお前の下には、こうも特徴的なヤツが集まるんだ？
ネロ : 「シュワシュワケトルXIV世」だぁ？
このオレ様の手にかかれば、そんなガラクタより、
よほど素早く簡単に湯を沸かす魔導機械が出来上がるぜ。
アルフィノ : やれやれ、今日は本当に妙な客ばかりが現れる日だな。
いずれも心強い協力者だ……と思いたいのだが……。
