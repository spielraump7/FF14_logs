鷹の巣

アルトアレール : ……では、改めて自己紹介させてもらおう。
我が名は、アルトアレール・ド・フォルタン。
フォルタン家の長子にして、誇り高きイシュガルドの騎士だ。
アルトアレール : 先ほどは失礼した……。
己に与えられた任務は、己の手で……
そう考えたゆえの言葉でな。
アルトアレール : 貴殿の実力は、あの男……オルシュファンから聞いている。
ぜひとも、その力を貸してくれ。
アルトアレール : ……といっても、力を貸す相手は、
小憎たらしい「デュランデル家」と「ゼーメル家」なのだがな。
アルトアレール : 四大名家のうち、デュランデル、ゼーメルの両家は、
我がフォルタン家とは、歴史的に対立してきた経緯がある。
ライバル関係にあると言えばいいだろうか。
アルトアレール : 今回は、教皇庁からの依頼で、
彼らが進めている事業の手伝いをすることになった。
癪ではあるが、せいぜい貸しを作ってやるとしよう。
アルトアレール : さあ、イシュガルド下層に向かうぞ。
任地はクルザス西部高地の「ファルコンネスト」……
チョコボ留の厩務員に声をかけ、現地までの足を手に入れよう。
アルトアレール : ここのチョコボ厩務員に声をかければ、
「ファルコンネスト」までの足を用意してくれるはずだ。
向こうで、また会おう。
チョコボ屋 ロドゥリック : Spielraum様でございますね？
アルトアレール様から、お話は伺っております。
貴方様のために、とびきりの黒チョコボを用意しておりますよ。
チョコボ屋 ロドゥリック : おや、黒チョコボをご存知ないので？
黒い毛並みが特徴的なフライヤー種という品種なのですが、
特別な訓練により、なんと空を舞うことができるのです。
チョコボ屋 ロドゥリック : 「ファルコンネスト」まで、氷壁を超えてひとっ飛び！
最初は、ちょっぴり怖いかもしれませんが、
緊張なさらずに、黒チョコボでの空旅をお楽しみください。
アルトアレール : 無事に着いたようだな。
ここと皇都を行き来する際は、黒チョコボを利用するといい。
アルトアレール : ともあれ、ここは冷える……。
ひとまず「ファルコンネスト」を指揮する、
「レッドワルド」卿のところに行こうじゃないか。
レッドワルド : ご無沙汰しております、アルトアレール卿。
応援に来ていただけると聞き、お待ちしておりましたよ。
……おや、そちらの御仁は？
アルトアレール : よろしく頼む、レッドワルド卿。
こちらは、我が家の客人、Spielraum殿……。
今回の応援要請に対し、力添えをしてくれることになった。
レッドワルド : おぉ、そうであったか！
私の名は「レッドワルド」、デュランデル家に仕える騎士だ。
貴公の活躍は、我が友、ドリユモン卿からも聞いているぞ！
アルトアレール : 我らふたりで、こちらの任務に助力させていただく。
何をなすべきか、担当を指示していただけるか？
レッドワルド : うむ、冒険者殿もいることだ。
少し背景事情も含めて説明させてもらおう。
レッドワルド : ……「第七霊災」以前、ここファルコンネストは、
哨兵の一隊が駐留するだけの寒村に過ぎなかった。
レッドワルド : だが、霊災を境に巻き起こった大寒波により、
雪と氷に閉ざされ、住民たちは皇都へ避難を余儀なくされた。
かくして西部高地は放棄されたのだが、その結果どうなったか。
アルトアレール : クルザス西部高地全体が、
「氷の巫女」率いる、異端者たちの根城と化した……。
レッドワルド : そのとおりだ。
教皇庁が西部高地の奪還を決定したのも、
活発化の一途を辿る異端者勢力への対処を進めるため……。
レッドワルド : ファルコンネストの再建は、その足掛かりだ。
だが、そうはいっても5年もの間、放棄されていたからな。
再拠点化には、大変な手間が掛かっている。
レッドワルド : 応援として来てもらった貴殿らには、
「各施設の再建作業」と「異端者勢力の捜索」……。
この二点について、協力してもらいたいのだ。
アルトアレール : 任されよう、レッドワルド卿。
フォルタン家の名にかけ、必ずや務めを果たす覚悟だ。
何なりと指示してくれ。
アルトアレール : 私は、異端者がらみの件で、
レッドワルド卿の手伝いをすることになった。
そちらの件は、任せたぞ。
エロワン : そこの貴方……。
もしや、フォルタン家のお客人では？
エロワン : やはり、そうでしたか！
実は一度、貴方にお会いして、
ご相談させていただきたいことがあったのです。
エロワン : この程、我ら神殿騎士団では、総長殿の発案による、
新たな取り組みを始めることになりました。
エロワン : それは、エオルゼア諸国の冒険者ギルドや、
グランドカンパニーのリーヴ制度を参考にしたもの……
いわば、「神殿騎士団リーヴ」といったところでしょうか。
エロワン : 熟練の冒険者である貴方にも、
ぜひ、この新制度に参加していただきたいのです。
報酬も用意しておりますので、どうかお引き受けください！
