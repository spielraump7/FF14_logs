聖竜の試練

アルフィノ : 聖竜の態度が変わっていないらしいというのは、

さあ、霊峰「ソーム・アル」を登り、雲海を目指しましょう。
アイメリク : 了解だ。
先導を頼むよ。

アルフィノ : それはよかった！

イシュガルドの教皇代行の立場にある者です。
アイメリク : 皇都の民を代表して、

フレースヴェルグ : 見え透いたことを……。

フレースヴェルグ : どうやら図星と見える……。
ならば聞くまでもなく、我が答えもわかっているはず……。
フレースヴェルグ : かつて、我が愛しきシヴァの想いが、竜と人との架け橋となった。

フレースヴェルグ : 血を分けたラタトスクを殺され、悲しみと怒りを抱いたのが、

すべての命を愛したシヴァの想いがあればこそ……。
彼女を喰らうときに立てた「不殺の誓い」ゆえのことよ。
フレースヴェルグ : 我が眷属に、己が生きる道を選ぶことを許した上で、
お主らには邪竜の影に動きありと、報せてやりもした……。
それ以上を望むのは、強欲がすぎるというものだ。
アルフィノ : 貴方が人の裏切りに絶望し、
静寂を望んでいることは承知しています。
アルフィノ : ですが、それでもなお、
絶望の中に、一片の希望を抱いておられるのではありませんか？
フレースヴェルグ : ほう、汝には、我が希望を抱いていると、
そのように見えると云うのか？
アルフィノ : はい……。
そうでないと言うのなら、希望を抱いていないと言うのなら、
なぜ貴方は、その背にイゼルを乗せたのですか？
フレースヴェルグ : イゼル……幻想を抱いた哀れな娘か……。
あれに背を貸したのは、己が罪を認めて悔い、
新たな未来へ進もうとする姿に、心動かされたゆえのこと。
フレースヴェルグ : 我が兄弟の眼を、悪しき人の手に、
委ねたままにしておくのも、忍びなかったのでな……。
アルフィノ : つまり貴方は、罪を認め悔いる人と、
罪を認めずあやまちを繰り返そうとする人とを、
わけて考えておられるということですね？
アルフィノ : 人という種そのものに絶望しているのではなく
あくまでも、どう過去と未来に向き合うのかに重きを置くと！
フレースヴェルグ : 詭弁を……。
だとしたら、どうだというのだ。
人の都を守るため、我に兄弟殺しの罪を被れと言うのか？
アイメリク : 血のつながりを理由に、罪を見逃すことは正しいことでしょうか？
取り返しの付かぬ罪を成そうとしているのであれば、
たとえ血を分けた肉親であろうと、止めてやらねばなりません。
アイメリク : かつて私は、ここにいる光の戦士殿に対し、
蛮神の力に手を染めた父を倒してくれと、そう頼みました。
父殺しの罪を犯したのも同然なのです……。
アイメリク : 己が手を汚していないぶん、より卑怯とさえ言えましょう。
ですが、それを成さねば多くの民が魂を焼かれ、
聖戦に投じられていた……。

ニーズヘッグは、我が友、エスティニアンの身体を奪いました。
影を払うことは友の死を意味しましょう。それでも私は……！
フレースヴェルグ : 多くの命を救うために、汝は友を殺すと？
ゆえに我にも、誓いを破り、兄弟を殺せと命じるのか？
ミドガルズオルム : 聞け、我が子よ。
少なくともハイデリンの使徒らの思惑は、異なるようだぞ。
皇都の民だけでなく、かの竜騎士の命をも救うつもりらしい。
ミドガルズオルム : そもそも「邪竜の影」が、ニーズヘッグそのものだと言えるのか？

何かを命じるつもりはないが、後悔せぬよう考えることだ……。
フレースヴェルグ : あくまで、あの竜騎士を救いたいと……。
イゼルがお主に希望を託したのも、その純粋さゆえか……。
だが、純然たる怨念ゆえに「影」の力は強いぞ？
フレースヴェルグ : どうやら見極めさせてもらう必要がありそうだな。
……お主ら、人の意思に「影」を払う力があるや否かを！
フレースヴェルグ : かつて我が兄弟、ラタトスクが棲処とした遺跡にて待つ。

アルフィノ : 聖竜は「試練の地」と言っていたね。

責任は重大だな……。

アイメリク : どんなに厳しい試練だったとしても、
皇都の未来のためには、耐え抜かねばならない。
Walt殿、互いに力を尽くそう。
アルフィノ : 聖竜は「試練の地」と言っていたね。

覚悟が決まったのなら、ただちに向かおうぞ……。
美しき翼の飛竜 : まさかフレースヴェルグ様が、試練を与えるとは……。

アイメリク : この試練…
突破してみせよう！
アルフィノ : ええ、必ず！

長老モグリン : みんな、サボるなくぽ！
