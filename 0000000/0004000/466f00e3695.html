ふたつの魂

フォルタン家の騎兵 : おや、Walt様ではありませんか。
アルフィノ様が、貴方様をお捜しの様子でしたよ。
フォルタン家の騎兵 : 屋敷内で帰りを待つとのことでしたので、
ぜひお立ち寄りのうえ、「アルフィノ」様にお声がけください。
フォルタン家の門衛 : お帰りなさいませ、お客人。
「フォルタン伯爵邸」に入られますか？
タタル : 「暁」のみなさんも、イシュガルドで良くしてくれた方も、
みんなで幸せな大団円にしたいでっす！
フォルタン家の家令 : 僭越ながら、ご報告申し上げますと……
目覚めたオノロワは、エマネラン様を案じて、
裸足で屋敷を出ていこうとした始末だったのでございます。
フォルタン家の家令 : まだまだ未熟な使用人ではございますが、
生涯尽くすにふさわしい、心の主を得ている様子……。
我らフォルタン家の使用人は、本当に幸せ者です。
アルフィノ : Walt……
少しばかり付き合ってもらいたいのだが、時間はあるだろうか？
アルフィノ : 先日行われた記念式典のことは覚えているね？
そう、ヴィゾーヴニルが襲撃された時のことさ。
アルフィノ : あの時、邪竜の影は、一時的に竜騎士の姿……
つまりエスティニアン殿の形をとって、襲撃を行った。
私は、ここにこそ、彼を助ける手がかりがあると思っているんだ。
アルフィノ : そこで、エーテル学に詳しいヤ・シュトラや、
クルルさんに助言を求めていたんだが……。
彼女たちから話したいことがあるそうでね。
タタル : ヤ・シュトラさんとクルルさんが、
イシュガルドまで来てくれているのでっす。
待ち合わせ場所の「忘れられた騎士亭」まで行くでっす！
アルフィノ : ありがとう。
それじゃあ、さっそく三人で出かけるとしようか。
クルルさんたちを、あまり待たせてはおけないからね。
アルフィノ : 待たせてしまったかな？
クルル : 平気よ、アルフィノくん。
ここのマスターさんが、スパイスワインを勧めてくれてね。
風味豊かな味わいを楽しんでいたところなの。
タタル : ジブリオンさんのスパイスワインは、
薬効で身体もホカホカになる、まさに絶品なのでっす！
ヤ・シュトラ : 確かに美味しかったわね。
……でも、ワインの品評よりも、
重要な話があるんじゃなくて？
クルル : そうね……それじゃ、本題に移りましょうか。
エスティニアンさんのことなんだけど……。
アルフィノ : 何かわかったのかい！？
クルル : ちょっと、焦らないで……。
まずは現時点で判明している情報から整理しましょう。
ヤ・シュトラ : 魔大陸での出来事を思い出してちょうだい。
確か、あなたはこう証言していたわね？
ヤ・シュトラ : エスティニアンが、ふたつの「竜の眼」を手にした直後、
突如として苦しみはじめ、邪竜の影と化してしまったと……。
アルフィノ : 確かに、ファルコンネストに現れた
エスティニアン殿の鎧には「竜の眼」らしき物が付いていた……。
ヤ・シュトラ : そんな生易しい状態ではなさそうよ。
私が視たのは、「竜の眼」から発する魔力が複雑に絡みつき、
エスティニアンのエーテルを、ほとんど覆い尽くす様だった。
アルフィノ : そんな！
エスティニアン殿は、完全にニーズヘッグと化してしまったと！？
クルル : だからそう、結論を急がないの！
ヤ・シュトラさんも言ったでしょう？
「ほとんど」って……。
クルル : 私も、あの場では圧倒されてしまったけど、
確かに「邪竜の影」には違和感を感じていたのよ。
クルル : でもヤ・シュトラさんの話を聞いて、確信したわ。
あの違和感の正体は、邪竜が放つ怨念の感情に沈殿した、
別の意思の痕跡なんじゃないかって……。
アルフィノ : それじゃあ……！？
ヤ・シュトラ : 私が視た、かすかなエーテルの輝き……
クルルが「言葉の壁を超える力」で感じた違和感……。
それは、エスティニアンの魂の存在を示している。
アルフィノ : 「竜の眼」を引きはがすことができれば、
エスティニアン殿を救えると！？
ヤ・シュトラ : それはわからないわ……。
ひとつの肉体に同居した人と竜のふたつの魂を、
分割したことなんて、ここにいる誰もがないのだから……。
アルフィノ : それでも希望があるなら、
私は諦めない……絶対にだ……。
クルル : アルフィノくん……。
クルル : 確かに希望はあるわ。
でも、注意してちょうだい。
クルル : 上手くいって「竜の眼」を切除できたとしても、
エスティニアンさんの魂が耐えられる保証はないわ。
引きはがした人が、新たな邪竜の影にされないとも限らない。
アルフィノ : 危険は承知の上さ。
辛抱強くチャンスを待ち、「竜の眼」を引きはがす！
あとはエスティニアン殿の魂の強さを信じるだけだ！
オノロワ : こちらにいらしたのですね、みなさん！
オノロワ : フォルタン伯爵邸に、
神殿騎士団よりの使いの方がいらっしゃいました。
アイメリク様が、お二方と面会したいとのことです、はい。
アルフィノ : アイメリク卿が？
……了解した。
わざわざ伝言を届けてくれて、ありがとう。
ヤ・シュトラ : それじゃ、私たちは任務に戻るわ。
アルフィノ : わかった。
こちらの件は任せてほしい。
ヤ・シュトラ、クルルさん、いろいろとありがとう。
アルフィノ : それじゃあ、先に神殿騎士団本部へ行っているよ。
クルル : アルフィノくん、
ちょっと焦ってるみたいだから……よろしくね？
ミドガルズオルム : あくまで友を救うつもりか……。
さて、どう動く、我が子らよ……。
アルフィノ : 来たね、Walt。
それにしても、いったいどんな用件なのだろうか？
気になるところだな……。
