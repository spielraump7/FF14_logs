決戦に備えて

アルフィノ : このタイミングでのアイメリク卿からの呼び出しとなると、
ドラゴン族がらみの話題だとは思うのだが……。
アルフィノ : はっ！？
もしやエスティニアン殿……
いや、邪竜の影の動向が明らかになったのでは？
アルフィノ : もしそうなら、こうしてはいられないぞ……。
Walt、さっそく総長室に向かおう！
アイメリク : 来てくれたか、ご両人！
アルフィノ : 邪竜の件で、何か動きがあったのですか？
アイメリク : いや……例の式典からというもの、
総力を挙げて捜索を行っているのだが、音沙汰なしだ。
ルキア : 竜騎士団から精鋭を選出し、
ドラヴァニア雲海の強行偵察を行わせたが、
成果はなかった……残念ながらな。
アルフィノ : そうでしたか……。
アイメリク : だが、邪竜自身が敵地に乗り込んでまで、
宣戦布告をした以上、近々動くつもりなのは間違いあるまい。
アイメリク : 我々としては、邪竜が動きを見せていない今こそ、
決戦に備えて準備を進めるだけだ。
アルフィノ : 邪竜を迎え撃つとなれば、
皇都の防備を固めねばなりませんね。
アイメリク : むろんだ。
四大名家を筆頭とする貴族たちの戦力に加え、
ヒルダたち平民の自警団も交えて、防衛計画を策定している。
アイメリク : だが、君たちを呼んだのは、その件ではない。
アルフィノ : と、申されますと？
アイメリク : いかに守りを固めようと、
両眼を取り戻した邪竜の力には及ぶまい。
対抗するには、同等の力が必要なのだ……。
アルフィノ : ……まさか！？
アイメリク : そうだ、聖竜「フレースヴェルグ」の力を借りたい。
アルフィノ : 確かに邪竜「ニーズヘッグ」と並ぶ存在となれば、
同じ七大天竜を置いて、ほかにありません。
しかし、かの竜は……。
アイメリク : わかっている……。
静寂を望む聖竜に、助力の意思がないということは、
エスティニアンからの報告でも聞いていたからな。
アイメリク : だが、たとえ僅かな可能性であっても、
何もせぬまま、諦めるわけにはいかないのだ。
どんな犠牲を払ってでも、民を守ると決意したのだから……。
アイメリク : ご両人、私は邪竜との決戦を、
教皇代行としての最後の務めであると考えている。
ゆえにどうか……不甲斐ない私に力を貸してくれ……。
アルフィノ : ……わかりました。
ご案内いたしましょう、聖竜の棲まう「白亜の宮殿」へ。
アルフィノ : 雲海のモーグリ族から託されたラッパがあれば、
ふたたび聖竜と会うことも適いましょう。
アイメリク : ありがとう、ご両人。
アイメリク : ルキア、留守を頼むぞ。
ルキア : ハッ！
どうか、お気を付けて……。アルフィノ : 聖竜フレースヴェルグか……。
厳しい交渉になるのは、想像に難くない。
が、エスティニアン殿のためにも、成功させなくては……。
アイメリク : 立て続けの案内と仲介の依頼となり、すまなく思う。
しかし、皇都の未来が懸かっているのだ。
改めてよろしく頼む……。
アルフィノ : 何をおっしゃいますか。
私もWaltも、皇都を訪れてから、
人と竜との争い、「竜詩戦争」に少なからず関わってきました。
アルフィノ : それに何より、ここイシュガルドには、
たくさんの友人と大切な仲間たちが暮らしているのです。
もう他人事ではありません。今更、余所者扱いは止してください。
アイメリク : アルフィノ殿……Walt殿……。
本当に、ありがとう……。
アルフィノ : さあ、それよりも行動を開始しましょう。
邪竜の影が、いつどのように動くかわからない以上、
時間を浪費するわけにもいきません。
アイメリク : そうだな……。
タタル嬢には、部下に事情を伝えるよう命じてあるが……
まずは、どちらに向かうつもりだ？
アルフィノ : 聖竜フレースヴェルグとの面会は、
通常、ドラヴァニア雲海の「白亜の宮殿」で行われます。
ですが、そこに向かう前に立ち寄りたい場所があるのです。
アイメリク : ほう……。
アルフィノ殿には、何か考えがあるようだな？
アルフィノ : はい……高地ドラヴァニアの「不浄の三塔」に立ち寄り、
聖竜の眷属、白き翼「ヴィゾーヴニル」に会おうと思います。
アルフィノ : 先日の式典で、かの竜は邪竜の影に襲われ手傷を負いました。
それを見舞いつつ、聖竜の動向について、
尋ねておこうと思うのです。
アイメリク : なるほど、名案だ。
では、さっそくだが高地ドラヴァニアに出発しよう。
