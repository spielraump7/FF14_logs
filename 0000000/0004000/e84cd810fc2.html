フォルタン家の人々

アルフィノ : Walt……
邪竜の影との激闘、よく戦い抜いてくれた。
勝利したこともだが、君が無事でいてくれて本当に嬉しいよ。
アルフィノ : 君がいなければ、エスティニアン殿を蝕んでいた、
邪竜の眼を引きはがすことなど、到底、不可能だったろう。
アルフィノ : ともかく今は、エスティニアン殿の容態が心配だ。
神殿騎士団本部に赴き、「ルキア」殿に確認してみよう。
アルフィノ : さあ、ルキア殿に聞いてみよう。
ルキア : Walt殿、アルフィノ殿……
よくぞ使命を果たし、聖竜を連れ帰ってきてくれた。
邪竜の影を退けてくれたことも、なんと感謝していいものか。
アルフィノ : 私がしたことなど、微々たるものです。
真の英雄は、邪竜の影を退けたWaltと、
影の意思をねじ伏せたエスティニアン殿のことでしょう。
アルフィノ : それで……エスティニアン殿の容態は？
ルキア : ……現在、神殿騎士団病院に搬送され、
病院長のエーベル卿が、直々に診察されている。
未だ意識は回復していないようだ。
ルキア : 長らく邪竜の影に、身体を乗っ取られていたのだ。
……ここは焦らずに見守るしかないだろう。
貴殿らはフォルタン伯爵邸に戻り、休息を取るといい。
アルフィノ : しかし！
ルキア : エスティニアン殿には、アイメリク様も付いておられる。
意識が回復したら、すぐさま連絡するゆえ安心してくれ。
何より、フォルタン伯爵らも心配しておられるだろう。
アルフィノ : ……わかりました。
アルフィノ : それでは、Walt……
ここは神殿騎士団に任せ、一度「フォルタン伯爵邸」に戻ろう。
ルキア : 今は、フォルタン伯爵邸に参られよ。
フォルタン家の方々を、安心させて差し上げるのだ。
フォルタン家の門衛 : お帰りなさいませ、お客人。
「フォルタン伯爵邸」に入られますか？
エドモン・ド・フォルタン伯爵 : おぉ、無事であったか！
アルフィノ : どうにか命拾いしました……。
我らが英雄の力が、邪竜を退けたのです。
そして最後は、亡き友たちに託された想いに助けられました。
アルトアレール : 私も、私なりに必死に戦ったつもりではあったが、
結局は英雄殿に、頼り切る形になってしまった……。
イシュガルドの騎士でありながら……不甲斐ない。
エマネラン : 全員が全員、英雄になれるわけじゃねぇってことさ。
ここは素直に、マブダチの偉業を称えようぜ！
イシュガルドのために戦ってくれて、ありがとよ！
エマネラン : あぁ、それからオレ様のことも褒めてくれよな！
第三区の対竜バリスタを指揮して、
竜どもをバッタバッタと撃ち落としたんだからよ！
オノロワ : 指揮といっても、大声で喚いただけです、はい。
ワイバーンを複数撃ち落としたのは確かですが、
褒められるべきは、射手の腕前でしょう。
エドモン・ド・フォルタン伯爵 : エマネランの手柄については、後ほど詳しく検証するとして、
ともかく、お二方ともお疲れでしょう。
エドモン・ド・フォルタン伯爵 : 部屋を用意させてあるゆえ、
ひとまず休息をとられるといい。
アルフィノ : 閣下……ご厚意は大変ありがたいのですが、
私は、もう一度、神殿騎士団本部へ向かおうと思います。
申し訳ありません……失礼します。
エドモン・ド・フォルタン伯爵 : 己の身体よりも、盟友のことが心配ということか。
アルフィノ殿も強くなられた……。
エドモン・ド・フォルタン伯爵 : どうやら貴殿も、ソワソワしておいでのご様子……。
さあ、我々のことなどよりも、蒼の竜騎士のもとへ、
見舞いに行くのがよろしかろう。
アルトアレール : 邪竜の影を退けることができたことは嬉しく思う。
だが、此度の防衛戦では、我が家の騎兵団にも多くの犠牲が出た。
遺族のことを考えると、諸手を挙げて喜んでばかりはいられない。
エドモン・ド・フォルタン伯爵 : Spielraum殿……
貴殿もエスティニアン殿の容態が気になっておられる様子。
さぁ、我らのことなど気になさる必要はありませんぞ。
エマネラン : 対竜バリスタを作ってるのは、
アインハルト家が管轄してるスカイスチール機工房だ。
そいつを活用して皇都を守ったってことは……ムフフフフ。
オノロワ : エマネラン様の果てしなく前向きなところだけは、
見習わなくてはならないのかもしれません、はい。
フォルタン家の執事 : イシュガルドのために、命がけで戦ってくださったこと、
私たちは決して忘れることはないでしょう。
フォルタン家の家令 : ご無事の帰還を心より喜んでおります。
本当にお疲れさまでした……。
ルキア : エスティニアン殿は、神殿騎士団病院に搬送した。
今や、病院こそが最前線……
多くの負傷者の治療が行われているのだ。
アイメリク : あぁ、君か……。
アルフィノ殿を探している、だと？
彼なら、私と交替でエスティニアンの看病をしているところだ。
アイメリク : せめて側にいるだけでもと迫られては、
断るわけにもいくまいよ。
アイメリク : 聖竜の試練を受けてから、取って返してのこの激戦……
アルフィノ殿も疲れているだろうに、ありがたいことだ。
エスティニアンは……いい友を持ったな。
