アルザダール海底遺跡群

ヤ・シュトラ : アルザダール3世……
異世界から帰ってきたという逸話を持つラザハンの偉人……
もしそれが本当なら……。
グ・ラハ・ティア : 案外、宝物殿に収められているお宝というのも、
鏡像世界に由来するものかもしれないな！
ウリエンジェ : 期待するのもほどほどに……
この地図が示すのが、いかなる場所なのか、
未だ確証はないのですから。
エスティニアン : ま、行ってみるしかないさ。
ここで想像を巡らせていても意味はないからな。
エスティニアン : 準備が良ければ、さっそく出港といこう。
目指すは「アルザダール海底遺跡群」だ。
マトシャ : あんた方の帰りを待つ間、
「富」を司るドグ神にお祈りしておくだす。
お宝がみつかるようにって……。
グ・ラハ・ティア : ここのところ事務作業ばかりで、体がなまってたんだよな。
ちょうどいい運動になりそうだ！
ヤ・シュトラ : 噂を鵜呑みにしたわけじゃないけれど、
「世界を飛び越える術」の糸口になる可能性があるなら、
行ってみる価値はあるわね。
ウリエンジェ : アルザダール3世の宝物殿と言うからには、
侵入者を撃退するための、罠があると考えられます。
警戒しつつ進みましょう。
エスティニアン : 挑んだ者が、誰ひとりとして帰ってこなかった海底遺跡……か。
何が待ち受けていたとしても、俺はただ、この槍を振るうだけだ。
グ・ラハ・ティア : 何だあれは……？
ウリエンジェ : ラザハン式の錬金窯のようにも見えますが……。
ヤ・シュトラ : 調べてみないことには、何とも言えないわね。
？？？？ : 待て！
その装置に触れてはならない！
グ・ラハ・ティア : あんたはヴリトラ……
いや、その姿のときはヴァルシャンだったか。
ヴァルシャン : ヴリトラと呼んでくれて構わない。
今さら隠す必要もないのでな。
ヴァルシャン : それにしても、
このような場所で君たちと再会することになろうとは。
よもや盗掘が目的……ではないだろう？
エスティニアン : 人聞きの悪いことを……と言いたいところだが、
強く否定はできんな。
ウリエンジェ : ……天竜ヴリトラよ。
あちらの装置についてお伺いしても……？
ヴァルシャン : ……かつて、私が錬金術師たちに作らせたものだ。
あるものを封印するためにな。
ヴァルシャン : まずは見てもらった方が早いだろう。
ヴァルシャン : 私にのみ与えられた権限で、一時的に封印を解いた。
ウリエンジェ : これは……ヴォイドクラック……！？
グ・ラハ・ティア : ……いや。
小さくともこれは……ヴォイドゲートだ！
ヤ・シュトラ : 何か手がかりがあればと思って来てみれば、
ヴォイドゲートだなんてね……。
エスティニアン : ヴォイドゲートとやらは、そんなに危険な代物なのか？
ウリエンジェ : シャーレアンにおいても、
深淵の扉を開くことは禁忌とされております。
その危険性ゆえに……。
ヴァルシャン : あれを見た君たちには、説明しておく必要があるだろう……。
グ・ラハ・ティア : 「ヴォイドゲート」と「ヴォイドクラック」は、
どちらも原初世界と第十三世界を繋ぐ次元の通路なんだが、
開き方によって、便宜的に呼び方を変えているんだ。
グ・ラハ・ティア : 次元の歪みで偶発的に生まれた裂け目が「ヴォイドクラック」、
何者かによって故意に開かれた扉が「ヴォイドゲート」……。
グ・ラハ・ティア : 封印してあるところを見ると、
あれは誰かが開いたヴォイドゲートと見て間違いないだろう。
規模は、それほど大きくないけどな。
グ・ラハ・ティア : 何にせよ、宝探しに来たはずが、
まさかあんなものを見つける羽目になるなんて……。
グ・ラハ・ティア : あれを見ていると、思い出すよ。
暗闇の雲との戦いでオレたちを救ってくれた、あのふたりのことをさ。
グ・ラハ・ティア : ヴリトラの話を聞きたいところだが……。
