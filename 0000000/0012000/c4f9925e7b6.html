新たなる冒険

タタル : おや、Sakさん、
ちょうどいいところに！
タタル : これからお茶を淹れるところでっしたので、
お急ぎでなければ、ご一緒しませんか？
タタル : では、すぐにご用意するのでっす！
タタル : 「暁」が解散してからというもの、
ここも、少し静かになりまっした……。
タタル : でもでも、寂しくなんかないのです……
タタル : このタタルの情報網によれば、皆さん、それぞれの目的のために、
忙しくしながらも、元気に過ごされているようでっすから！
タタル : かく言う私も、
多少は時間に融通が利くようになりまっしたので、
あれをやろう、これをやろう……といろいろ考え中なのでっす！
タタル : 冒険者さんは、これからどうされるのでっすか？
タタル : フフフ、天の果てまで行ってきたというのに、
冒険者さんは相変わらずでっすね！
それなら、私に良い考えが……！
タタル : これまで冒険者さんは、
困っている方々を助けたり、世界の脅威に抗ったりするために、
戦い続けてきまっした。
タタル : そうして「英雄」と呼ばれるまでになりまっしたが……
終末を払ったいま、その肩書はいったん置いてしまっても、
良いと思うのでっす。
タタル : そして、初めて「砂の家」でお会いした日のように……
ひとりの冒険者に戻って、世界を巡ってみてはいかがでっすか？
タタル : この世界の広さを考えれば、
無理からぬことなのでっす……。
タタル : でもでも、ご安心を！
このタタルに良案があるのでっす……！
タタル : その目的地とは……ずばり、サベネア島！！
タタル : 前回は、終末の騒動もあって、
ラザハンの都も、ゆっくり観光できていないはずでっす。
ヴリトラさんのためにも、ぜひ！
タタル : それに、サベネア島が浮かぶ豊穣海は、
最近まで帝国の影響力が強く、
冒険の手も及んでいなかったんだとか。
タタル : きっと、今も多くの謎や神秘が眠っているはずでっす！
エメトセルク : ……お前、豊穣海に沈む海底遺跡に行ったことがあるか？
エメトセルク : ……アゼムのクリスタルを持つ者だろう？
それくらいは当然見ておけ。
タタル : どうやら目的地が決まったようでっすね！
新たな冒険を前に、ワクワクが止まらないといった顔でっす！
タタル : そんな冒険者さんの門出を祝して、
私から新たな旅装をプレゼントしたいのでっすが……
実はまだ作りかけでっして。
タタル : 大急ぎで完成させまっすので、
もう少し、このままお待ちいただけまっすか？
タタル : あれがこれでこうなって……
それがどれでああなって……
タタル : すぐに完成させるので、
少～しだけ待っていてほしいのでっす！
タタル : フフ、フフフフフ！
ノってきたでっす～！
この調子でバリバリ縫っていくのでっす～！
タタル : 見えない部分にも、ポッケをたくさん用意しておくのでっす……。
冒険者さんは使いもしないのに、
毛皮や羽根や葉っぱなんかを拾ってくる人でっすから……。
タタル : それから撥水性も大事でっす……。
海だろうと川だろうと温泉だろうと、
お構いなしに服のまま入ってしまう人でっすからね……。
タタル : 目が覚めまっしたか？
タタル : お待たせしてごめんなさいでっす！
つい、針がノってしまい……。
タタル : なにか、いい夢でも見まっしたか……？
それとも、未知の冒険への興奮が漏れ出ているのでしょうか。
タタル : そんな心躍ってらっしゃる冒険者さんに、
ぴったりの旅装が、もうすぐ完成しまっす！
タタル : 仕上げは工房で行うので、
「ダイヤモンドフォージ」までご足労いただけまっすか？
タタル : では、ドドーンと仕上げるのでっす……！
タタル : さっきのように、長くお待たせすることはありまっせんから！
タタル : 肩幅をもう少し……
それでもって、袖口はっと……
最後にズバババッと、裾を合わせて……
タタル : ……これで完成！
さっそくご覧いただきたいのでっす！
タタル : 刮目してご覧あれ……
これがSakさんの、
新たなる冒険の幕開けを飾るのに、ふさわしい旅装なのでっす！
タタル : お気に召していただけたら、頑張った甲斐があるのでっす！
タタル : お代なんていいのでっす！
タタル : それよりも、ぜひともこれを着て、
いろいろなところを旅してほしいのでっす！
タタル : そう、人目につけばつくほど都合が良いのでっす……
Sakさんに憧れる、
多くの冒険者さんたちの目にとまるように……。
タタル : きっとみなさんこう思うはずでっす……。
「あの風格ある冒険者の装束は、どこで手に入るんだ？」と……。
タタル : やがて、この装備を求めて多くの冒険者さんたちが、
私の「大繁盛商店」に押しかけてくるに違いないのでっす！
タタル : ふおおおおお！
ビッグビジネスの予感がするのでっすー！
タタル : ……そうそう、新たなる冒険を機に、
これまで出会った人たちや、出来事について、
振り返ってみるのはいかがでっすか？
タタル : 記録として残しておけば、
何かの役に立つかもしれないでっすよ！
