サベネア島の今

ヴァルシャン : 宝の使い道について相談する上で、
まずは民の声を広く聞きたい。
ヴァルシャン : 太守である私には話しにくいという者もいるだろうから、
君たちも聞き取りに協力してくれないだろうか？
ヴァルシャン : ありがとう。
マトシャたちのような漁師や、商人たちに、
困ったことがないか聞いてみてくれ。
ヴァルシャン : エスティニアンは、パーラカの里を頼む。
私はギガントガル採石場を回ってくるつもりだ。
ヴァルシャン : 各々、聞き込みが終わったら、
ラザハンのメーガドゥータ宮へ……。
よろしく頼むぞ。
マトシャ : あんたが無事に帰ってきてくれて、安心しただす。
船はもういいだすか？
マトシャ : オラたちの生活で、困ったことがないか……？
マトシャ : うーん、そうだなぁ。
やっぱり、カルザールさんがいなくなったのが、
大きく響いているだす……。
マトシャ : オラは、新しい販路を開拓して、
少しは魚を売れるようになってきてるけど……
それでも昔に比べたら、まだまだ大変だす。
マトシャ : 今は村のみんなで助け合ってるけど、
先を考えると、不安になることも多いだす。
よろず屋 : いらっしゃい！
あんた、外からのお客さんだね！
サベネア島に来たなら、こいつを飲まなきゃモグリってもんだ！
よろず屋 : 「特製アームララッシー」、3つで1万9800ギルのところ、
今なら1万ギルポッキリ！
さあ、買った買った！
よろず屋 : エスティニアンってのは、どちらさんで？
って……。
よろず屋 : あんたは、槍使いの男と一緒にいた……！
どうやらぼったくりだと、気づいてるようだな……。
よろず屋 : だってよぉ、しょうがねえじゃねえか……。
終末が終わったばかりで、どの国も自国の立て直しが優先なんだ。
この島まで足を伸ばす観光客なんざ、そう多くはない……。
よろず屋 : 地元の人間だって余裕がなくて、適正価格ですら売れねえ。
冒険者相手にぼったくりの商売でもしなきゃ、
今のこの街じゃ、満足に食っていくことはできねえんだ……。
港湾ギルドの職員 : はぁ……。
港湾ギルドの職員 : この街の生活ぶりについて聞きたいって？
港湾ギルドの職員 : 終末の終息に伴って交易が再開され、
景気が上向きつつあるのは確か。
港湾ギルドの職員 : でも、交易商たちの中には、目先の生活のため、
商売道具である船を手放してしまった人も、少なくない。
港湾ギルドの職員 : 終末で命を落とした船乗りだってたくさんいたし、
以前と比べて、輸出量は激減……。
港湾ギルドの職員 : カルザール商会が残っていれば、
精力的に復興の音頭をとってくれたんでしょうけど……
もっと、あの人の力になれていれば……。
エスティニアン : 聞き込みのために歩いていたら、
何度も島民たちに呼び止められては、礼を言われてな。
ヴリトラの背に乗って戦う姿を、見られていたようだ。
ヴァルシャン : おかえり。
そちらも終わったようだな。
エスティニアン : お前、その人形のままで聞き込みに行ってきたのか？
今さら正体を隠すでもなし、
本体で移動した方が早そうなものだが……。
ヴァルシャン : 突然、空から巨体の竜が舞い降りては、民たちが驚くだろう？
何より、私の身体が収まる場所は、そうないのでな。
エスティニアン : フッ……違いない。
エスティニアン : さて、まずは俺から報告するとしよう。
エスティニアン : パーラカの里は、どうにか持ち直してきた様子だった。
エスティニアン : 避難民が多かった時期は、苦しい生活を強いられていたようだが、
イェドリマンの連中の支援もあって、耐え抜くことができたそうだ。
エスティニアン : 元々は、森と川での狩猟によって糧を得て、
自給自足に近い生活を送っていたようだからな。
万全ではなくとも、なんとか生きていけるとのことだ。
ヴァルシャン : ふむ……。
パーラカの里は、困窮してるとまではいかないようだな。
ヴァルシャン : では、次は私から報告しよう。
ヴァルシャン : ギガントガル採石場では、久しぶりに入港した異国の商船に、
巨人石が売れたとのことで、持ち直しの兆しが見られる。
ヴァルシャン : しかし、ラザハンの商人による石材の輸出は、
最盛期と比べると、遠く及ばないようだ。
ヴァルシャン : ここラザハンは、島国ということもあって、
収入の大半を、他国への輸出貿易によって得ている。
ヴァルシャン : 交易が滞っている理由がわかれば、対処のしようもあるが……
アキャーリとイェドリマンは、どんな状況だったか聞かせてくれ。
ヴァルシャン : なるほど……。
アキャーリでは、漁で得た魚の卸先が少ないために、
日々の稼ぎに限界があるようだな……。
ヴァルシャン : そのためには、イェドリマンが抱えている、
交易船と人手不足の問題の解決が急務のようだな。
ヴァルシャン : ここに来るまでの道中、街道をひとりで歩く子どもを見かけた。
終末で船乗りだった親を亡くしたのだそうだ……。
ヴァルシャン : それだけ、多くの命が失われたということか……。
ヴァルシャン : いや、ここで嘆いていても仕方がないな。
これより私は、君たちの聞き込みによって得た情報をまとめ、
困窮した者たちを救う方法を考えよう。
ヴァルシャン : 方針が決まり次第、皆を集めて指示を出す。
君たちが集めた民の声に、私がどう応えるのか。
最後まで見届けてほしい。
ヴリトラ : 皆、集まってくれて感謝する。
ヴリトラ : 先ほど、星戦士団に運んでもらった金銀財宝は、
アルザダールの名を借りて、私が有事に備えて蓄えてきたものだ。
ヴリトラ : それを投じるべきは、今を置いてほかにあるまい。
終末により傷ついた多彩なる都に、彩りを取り戻すために……！
ヴリトラ : 常ならば、このようなとき、
私は太守たちに頼り、その仕事にまかせてきた。
だが、アヒワーン亡き今、彼の手腕に頼ることはできない。
ヴリトラ : まず、宝物殿より運び出した財宝の換金を進め、
これを財源として、イェドリマンの交易商に出資すべし！
人手と船を失った商人を助け、交易の活性化を促すのだ……！
ヴリトラ : むろん、あれほどの惨劇を経験したのだ。
誰もが支援を必要としていることは重々承知している。
ヴリトラ : だが、交易商への支援は、
カルザール商会という大きな卸先を失った、
漁師や職人、石工たちを救うことにも繋がると信じている。
エスティニアン : 終末で親を失った子どもたちへの対応は、
どうするつもりだ？
ヴリトラ : その点は、私も懸念するところだ。
ただ単に金を施すだけでは、いずれ資金は枯渇し、
孤児たちがふたたび路頭に迷うことになってしまう……。
ヴリトラ : 子どもたちを恒久的に保護していくための、
新たな施策を考えなければならないと思っている。
ヴリトラ : そこで、冒険者として数多くの国を見てきた、
君の意見を聞きたいのだが……。
ヴリトラ : なるほど……それは名案だ。
ヴリトラ : 素晴らしいアイディアだ。
出資の条件として、商人たちに売上の一部を、
孤児院運営に拠出するよう契約を結ぶとしよう。
ヴリトラ : さて、そうとなればこの制度にも、名をつけたいところだが……
ヴリトラ : 偉大な交易商……？
なるほど、カルザールのことか……！
ヴリトラ : サベネア島を恐怖に陥れた偽神獣としてではなく、
ラザハンの偉大な交易商として、彼の名を遺そうというのか……。
ヴリトラ : 異論などあるはずもない……。
しかし、この中にも獣に家族を奪われた者は多い。
皆の意見も聞かせてくれ。
ナブディーン : あのときは多かれ少なかれ、みんなが動揺していた。
誰もが獣になる可能性があったんです。
ナブディーン : カルザールさんを救えなかった悔しさはあれど、
恨んでなんかいませんよ！
ナブディーン : みんなだってそうだろ！？
エスティニアン : あのガキ、メラドとか言ったか。
これで、あこぎな商売から足を洗えるかもな。
ヴリトラ : では、ここに「カルザール基金」の立ち上げを宣言する！
善は急げだ、すぐさま取り掛かるとしよう！
ヴリトラ : ナブディーンは隊を編制し、
孤児たちを保護するべく巡回へ向かってくれ！
ヴリトラ : 君たちは、直ちに基金の運用を開始してくれ！
一刻も早く、困窮した民を救うのだ！
エスティニアン : 竜と人が手を取り合って、立ち上がろうとしている……
思い出すな、イシュガルドを。
エスティニアン : ヴリトラの奴、表立って太守然とするのにも慣れてきたじゃないか。
伊達に長生きしちゃいないな。
